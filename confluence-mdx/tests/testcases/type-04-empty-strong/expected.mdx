### 메뉴 설명

* [ **General** ](#unexpected-failure) **:** 쿼리파이의 기본 시스템 설정을 적용하는 메뉴입니다. 
* [ **Security** ](#unexpected-failure) : 쿼리파이의 보안 설정을 적용하는 메뉴입니다. 
* [ **Allowed Zones** ](#unexpected-failure) **** : 쿼리파이의 허용 네트워크 존 설정을 구성하는 메뉴입니다. 
* [ **Channels** ](#unexpected-failure) : 쿼리파이의 알림을 받을 채널을 구성하는 메뉴입니다.
* [ **Alerts** ](#unexpected-failure) : 쿼리파이 알림을 설정하는 메뉴입니다. 
* [ **Licenses** ](#unexpected-failure) : 쿼리파이의 적용 라이선스 현황을 조회할 수 있는 메뉴입니다.
