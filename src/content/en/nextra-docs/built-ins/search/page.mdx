# `<Search>` Component

|                 |                               |                                       |                                                                                 |
| --------------- | ----------------------------- | ------------------------------------- | ------------------------------------------------------------------------------- |
| `emptyResult`   | `string \| ReactElement{:ts}` | `'No results found.'{:ts}`            | Not found text.                                                                 |
| `loading`       | `string \| ReactElement{:ts}` | `'Loading…'{:ts}`                     | Loading text.                                                                   |
| `errorText`     | `string \| ReactElement{:ts}` | `'Failed to load search index.'{:ts}` | Error text.                                                                     |
| `placeholder`   | `string{:ts}`                 | `'Search documentation…'{:ts}`        | Placeholder text.                                                               |
| `className`     | `string{:ts}`                 |                                       | CSS class name.                                                                 |
| `searchOptions` | `PagefindSearchOptions{:ts}`  |                                       | `{preload?: boolean; verbose?: boolean; filters?: object; sort?: object}{:ts}`. |
