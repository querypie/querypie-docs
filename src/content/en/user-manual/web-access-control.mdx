# Web Access Control

### Overview

QueryPie WAC is a solution that controls and monitors access to web applications. It enhances security by applying role-based access control (RBAC) to all web applications, including internal management sites and SaaS platforms. It provides complete visibility by tracking and recording all user web activities in real-time.
