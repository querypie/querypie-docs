## Overview

QueryPie provides functionality to control access permissions to Kubernetes clusters and record user activities who access Kubernetes clusters as audit logs. You can access clusters with granted permissions through the QueryPie agent, and the KUBECONFIG file required for cluster access is configured through the agent.
