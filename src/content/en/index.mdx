# Welcome to QueryPie Product Documentation

## What is QueryPie?

QueryPie is an integrated security solution that provides **DAC, SAC, KAC, WAC** in a single unified platform. It covers both cloud and on-premises environments, providing unified management of all access from databases to systems, Kubernetes, and web applications.

It enhances enterprise data security through RBAC/ABAC-based dynamic access control, web SQL editor, real-time auditing, and anomaly detection alerts. With Docker-based easy installation and web-based interface, it provides a hybrid solution that combines the convenience of SaaS with the powerful security of on-premises.

**QueryPie Community Edition** has been released! Experience all the features of the Enterprise version for free.

It includes all access control features of DAC, SAC, KAC, WAC, and can be operated with up to 5 user accounts.
You can get started immediately within 7-10 minutes with Docker-based simple installation.
Automatic installation is possible with a one-line command `bash <(curl https://dl.querypie.com/setup.v2.sh)`, and provides a free license valid for 1 year.
When the validity period expires, you can extend the Community Edition license for free.

* **[QueryPie Community Edition Installation Guide](https://www.querypie.com/en/resources/learn/documentation/querypie-install-guide)** - Free installation and trial guide
