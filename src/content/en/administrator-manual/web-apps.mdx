## Overview

QueryPie WAC (Web App Access Control) is a Web Application protection solution for access control and logging of web applications.

The detailed features provided by WAC are as follows:

* Access Control: Allows access to web applications only for authorized users.
* Audit Logs: Records the history of work performed by accessing web applications even for authorized users.

Additionally, through feature updates, the following features are planned to be provided:

* Permission Control: Controls the actions of authorized users for critical tasks in web applications.
* Sensitive Information Control: Restricts the display of important information even for authorized work by authorized users.

## Web Apps Sub-features

To access the WAC management menu, click Web Apps in the top menu of the QueryPie Admin page.

Brief descriptions of each sub-menu are as follows:

* Connection Management
    * Web Apps: Register and manage web apps to be controlled.
* Web App Access Control
    * Access Control: Grant or revoke roles to QueryPie groups and users.
    * Roles: Manage Web App roles. You can create/modify/delete roles, view users and allowed web apps by role, assign/revoke policies, etc.
    * Policies: Manage Web App policies. You can create detailed policies such as allowing/blocking access to web apps, view roles assigned to those policies, and check policy modification history.
