## Overview

You can pre-register accounts that can access connections and store them in template form. Templates created in advance can be easily and conveniently loaded in servers or server groups.


## Creating Server Account Templates

<figure data-layout="center" data-align="center">
![Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templates &gt; Create Template](/administrator-manual/servers/server-account-management/server-account-templates/image-20240828-011910.png)
<figcaption>
Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templates &gt; Create Template
</figcaption>
</figure>

1. Navigate to Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templates menu.
2. Click the `Create Template` button in the top right. 
3. Enter the following information for template creation. 
    1.  **Name** : Enter a name to distinguish the template. 
    2.  **Description** : Enter content that can describe the template. 
4. Save through the `Save` button. 
5. Click the newly created template from the list. 
6. Click `Add Account` in the Account item to add accounts to register in the template. 
    * For individual Account setup methods, refer to [Managing Servers as Groups](../connection-management/server-groups/managing-servers-as-groups). 
7. Click `Save Changes` in the top right to save.
