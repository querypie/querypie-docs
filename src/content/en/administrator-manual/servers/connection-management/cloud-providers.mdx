# Cloud Providers

### Overview

QueryPie supports integration with Cloud Providers for server registration and management. You can synchronize resources within Cloud Providers, register them as servers managed by QueryPie, and grant access permissions and set policies for users and groups on the synchronized servers.

### Viewing Cloud Providers

1. Navigate to Administrator &gt; Servers &gt; Connection Management &gt; Cloud Providers menu.
2. You can search by cloud provider name using the search box in the top left of the table.
3. You can refresh the Cloud Provider list using the refresh button in the top right of the table.
4. The table provides the following column information:
    1.  **Name**  : Cloud provider name
    2.  **Cloud Provider**  : Connected platform
    3.  **Region**  : Platform region
    4.  **Replication Frequency** : Synchronization cycle setting
    5.  **Created At**  : Cloud provider initial creation date and time
    6.  **Updated At** : Cloud provider last modification date and time
5. Click on each row to view cloud provider detailed information.


### Creating Cloud Providers

1. Navigate to Administrator &gt; Servers &gt; Connection Management &gt; Cloud Providers menu.
2. Click the `+ Create Provider` button in the top right.
3. Create a cloud provider by referring to the following guides:
    1. [Synchronizing Server Resources from AWS](cloud-providers/synchronizing-server-resources-from-aws)
    2. [Synchronizing Server Resources from Azure](cloud-providers/synchronizing-server-resources-from-azure)
    3. [Synchronizing Server Resources from GCP](cloud-providers/synchronizing-server-resources-from-gcp)


### Deleting Cloud Providers

<figure data-layout="center" data-align="center">
![image-20240902-044732.png](/administrator-manual/servers/connection-management/cloud-providers/image-20240902-044732.png)
</figure>

1. Navigate to Administrator &gt; Servers &gt; Connection Management &gt; Cloud Providers menu.
2. Check the checkbox on the left of the target cloud provider to delete in the table.
3. Click the `Delete` button that appears in the table column line.
4. When a popup appears, click the `Delete` button to proceed with deletion.

<Callout type="important">
Please note that when deleting a Cloud Provider, all resources synchronized through that cloud provider will be bulk deleted together.
</Callout>
