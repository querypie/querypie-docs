## Overview

QueryPie SAC can centrally manage permissions for various systems and servers in cloud and on-premises environments, network equipment, and all resources accessible via SSH. Additionally, you can group numerous servers to easily and conveniently set access policies and grant permissions. It provides advanced features for real-time monitoring of user system access status and session recording capabilities that allow you to replay work history on the same screen.

## Recommended SAC Initial Setup Order

For administrators new to SAC (System Access Controller), we recommend following the setup order below for smooth server management. Setting up essential resources like Account and SSH Key first, which are required for server registration, will help you complete the work faster and more conveniently.
