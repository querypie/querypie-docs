# AWS Athena Specific Guide

### Manual Connection Creation

Select Administrator Page &gt; Databases &gt; Connection Management &gt; DB Connections &gt; Create Connections &gt; Athena. Then enter the required values. When creating manually, you must enter AWS Access Key and AWS Secret Key. This information must use information registered in IAM. You can check this in AWS IAM &gt; Users page.

### Cloud Provider

For Athena connections synchronized through Administrator Page &gt; Databases &gt; Connection Management &gt; Cloud Provider, separate AWS Access Key and AWS Secret Key input is not required.
