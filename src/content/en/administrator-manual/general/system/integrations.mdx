## Overview

The Integration menu provides easy support for integration with external security services. Through integration with various security tools, you can enhance system security, reduce system complexity, and simplify management and maintenance. As of version 10.0.0, SIEM/SOAR and Secret Store integration are supported, and the supported SIEM services are Syslog and Splunk.

<Callout type="info">
**What is SIEM?** <br/>SIEM (Security Information and Event Management) supports real-time collection and analysis of event log data from multiple sources. It can analyze log correlations, identify abnormal activities, and take appropriate actions.
</Callout>
