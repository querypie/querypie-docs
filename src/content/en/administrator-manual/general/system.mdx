# System

### Overview

This is a menu for managing QueryPie system settings.
You can manage integration with external security services, External API Tokens, and tasks that run periodically within QueryPie in each submenu.

### Submenus

Each menu provides the following functions.

* [Integrations](system/integrations) : SIEM service and Secret Store integration management, Notification, Authentication, and AI Chat management
* [API Token](system/api-token) : External API Token management
* [Jobs](system/jobs) : List of periodic/manual tasks within QueryPie
* [Maintenance](system/maintenance) : Maintenance mode management for zero-downtime or brief-downtime upgrades

Please refer to the descriptions of each submenu for detailed information.
