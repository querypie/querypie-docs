# QueryPie v10 Manual

### What is QueryPie ACP?

QueryPie ACP is a unified security solution offering  **DAC, SAC, KAC, WAC**  across cloud and on-premises environments.
It manages access consistently across databases, systems, Kubernetes, and web applications.

#### Core Product Components

* [ **<u>DAC (Database Access Controller)</u>** ](https://www.querypie.com/en/platform/security/database-access-controller) **-** Database access control and auditing
* [ **<u>SAC (System Access Controller)</u>** ](https://www.querypie.com/en/platform/security/system-access-controller) **-** System and server access control
* [ **<u>KAC (Kubernetes Access Controller)</u>** ](https://www.querypie.com/en/platform/security/kubernetes-access-controller) **-** Kubernetes cluster access control
* [ **<u>WAC (Web Access Controller)</u>** ](https://www.querypie.com/en/platform/security/web-access-controller) **-** Web application access control

It enhances enterprise data security through RBAC/ABAC-based dynamic access control, web SQL editor, real-time auditing, and anomaly detection alerts.
With Docker-based easy installation and web-based interface, it provides a hybrid solution that combines the convenience of SaaS with the powerful security of on-premises.
