## 커넥션 수동 생성
관리자 페이지 > Databases > Connection Management > DB Connections > Create Connections > Athena를 선택합니다. 이후 필수인 값들을 입력합니다. 수동 생성시 AWS Access Key와 AWS Secret Key를 입력해야 합니다. 이 정보는 IAM에 등록된 정보를 사용해야 합니다. AWS IAM > Users 페이지에서 확인할 수 있습니다.

## Cloud Provider
관리자 페이지 > Databases > Connection Management > Cloud Provider를 통해 동기화 Athena 커넥션에서는 별도의 AWS Access Key와 AWS Secret Key 입력이 필요하지 않습니다.
