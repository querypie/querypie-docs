## 커버로스 키탭 등록하기
Kerberos Configurations 페이지 우상단 Create Keytab 버튼을 클릭합니다. 키탭 파일을 올리고, 커버로스 구성의 이름을 입력한 후 Save 버튼을 클릭하여 저장합니다.

Administrator > Databases > Connection Management > Kerberos Configurations > Create Keytab

## 커버로스 키탭 조회하기
Kerberos Configurations 페이지에서 등록된 키탭 목록을 조회할 수 있습니다.

Administrator > Databases > Connection Management > Kerberos Configurations

## 커버로스 키탭 삭제하기
Kerberos Configurations 페이지 내 목록에서 삭제하려는 아이템을 체크박스로 선택하면 Delete 버튼이 표시됩니다. 버튼을 클릭하고, 확인 모달에서 Delete 버튼을 클릭하면 삭제가 완료됩니다.

Administrator > Databases > Connection Management > Kerberos Configurations > Delete Keytab

## 커버로스 클라이언트 구성 변경하기
Kerberos Configurations 페이지 우상단 ⚙️ 버튼을 클릭하여 QueryPie에 적용할 커버로스 클라이언트 구성 변경 모달을 노출합니다. 변경 내용을 저장하려면 Save 버튼을 클릭하세요.

Administrator > Databases > Connection Management > Kerberos Configurations > Kerberos Configuration Setting
