## Overview
QueryPie의 시스템 설정을 관리하는 메뉴입니다. 각 하위 메뉴에서 외부 보안 서비스와의 연동, External API Token, QueryPie 내부에서 주기적으로 실행되는 작업을 관리할 수 있습니다. 

## 하위 메뉴
각각의 메뉴는 다음과 같은 기능을 제공합니다.

* Integrations : SIEM 서비스 및 Secret Store 연동 관리
* API Token : External API Token 관리
* Jobs : QueryPie 내부의 주기적/수동 작업 일람

자세한 내용은 각 하위 메뉴에 대한 설명을 참고해 주세요.
