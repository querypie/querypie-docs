## Overview
Data Discovery의 수행결과에 대한 전반적인 내용을 시각적인 표현(위젯)을 통해 확인할 수 있습니다.각 위젯에 데이터가 표시되기 위해서는 최소 1회의 Discovery job이 성공적으로 수행되어야 합니다.

## 위젯
* Discovery Summary : Discovery Job의 수행 결과 및 Verification 의 결과를 수치로 요약 표시합니다.Scanned Database : Discovery Job이 정상적으로 수행되어 확인된 Database 의 총 개수 입니다.Databases with Sensitive Item : 관리자가 verification을 한 결과 sensitive item이 존재 database의 총 개수 입니다. scan이 완료되어도 관리자가 verification을 수행하지 않으면 이 값은 변하지 않습니다.Scanned Column : Discovery Job이 정상적으로 수행되어 확인된 Column의 총 개수 입니다.Columns with Sensitive Item : 관리자가 verification을 한 결과 sensitive item이 존재하는 column의 총 개수입니다. scan이 완료되어도 관리자가 verification을 수행하지 않으면 이 값은 변하지 않습니다. * Sensitive Items to Verify : Discovery Job이 정상적으로 수행되어 관리자가 verify 해야할 대상의 현황을 표시합니다. 관리자가 모든 대상의 verification 을 마치면 이 위젯은 데이터를 표시하지 않습니다.  Confidence : 발견된 항목의 95%신뢰구간 수치를 계산하여 Low / Medium / High 의 3단계로 표시합니다. 정규식의 경우(이항분포)의 예를 들자면, 발견된 항목이 100개 중 32개 이하면 Low, 33~66개면 Medium, 67개 이상일 경우 High입니다. 이 수치로 AI identifier 또는 정규식의 검출 수준을 판단할 수 있습니다.
