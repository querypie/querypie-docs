## Overview
현재 실행 중인 쿼리를 조회할 수 있습니다. 쿼리 실행이 종료되면 Query Audit에 내용이 이력으로 기록됩니다.

10.3.0에서 Audit > Databases > Running Queries 에 있언 Running Quries 메뉴가 Databases > Monitoring > Running Queries 로 이동되었고 Web Editor, Workflow를 통해 수행한 쿼리의 중지기능이 추가되었습니다. 자세한 내용은 링크의 문서를 참고 바랍니다.

## Running Queries 조회하기
Administrator > Audit > Databases > Running Queries

1. Administrator > Audit > Databases > Running Queries 메뉴로 이동합니다.
2. 당일 기준으로 내림차순으로 로그가 조회됩니다.
3. 테이블 좌측 상단의 필터 버튼을 클릭하여 AND/OR 조건으로 이하의 필터링이 가능합니다.SQL Type : SQL 구문 유형Action At : 쿼리 실행 일시 범위Executed From : 쿼리 실행 주체Web Editor : 쿼리파이 웹 에디터에서 실행 Proxy : 쿼리파이 프록시를 통한 실행SQL Request : SQL Request 워크플로를 통한 실행SQL Job : SQL Job에 의한 실행SQL Export Request : SQL Export Request 워크플로를 통한 실행
  1. SQL Type : SQL 구문 유형
  2. Action At : 쿼리 실행 일시 범위
  3. Executed From : 쿼리 실행 주체Web Editor : 쿼리파이 웹 에디터에서 실행 Proxy : 쿼리파이 프록시를 통한 실행SQL Request : SQL Request 워크플로를 통한 실행SQL Job : SQL Job에 의한 실행SQL Export Request : SQL Export Request 워크플로를 통한 실행
    1. Web Editor : 쿼리파이 웹 에디터에서 실행
    2. Proxy : 쿼리파이 프록시를 통한 실행
    3. SQL Request : SQL Request 워크플로를 통한 실행
    4. SQL Job : SQL Job에 의한 실행
    5. SQL Export Request : SQL Export Request 워크플로를 통한 실행
4. 테이블 우측 상단의 새로고침 버튼을 통해 로그 목록을 최신화할 수 있습니다.
5. 테이블에서 이하의 컬럼 정보를 제공합니다:No : 이벤트 식별 번호Executed At : 쿼리 실행 일시Name : 대상 사용자Email : 대상 사용자 이메일Action Type : 액션 유형Connection Name : 대상 DB 커넥션명Database Type : 대상 데이터베이스 유형Privilege Name : 사용자 접속 권한Client IP : 사용자 클라이언트 IP 주소Replication Type : DB Replication 유형DB Host : 접속한 DB 호스트DB Name : DB명DB User : DB 사용자 IDTable(s) : 호출 테이블명SQL Type : SQL 구문 유형Query : 실행 쿼리문Client Name : 이용 클라이언트명 (DataGrip 등)Execution Reason : 쿼리 실행 사유Executed From : 쿼리 실행 주체
  1. No : 이벤트 식별 번호
  2. Executed At : 쿼리 실행 일시
  3. Name : 대상 사용자
  4. Email : 대상 사용자 이메일
  5. Action Type : 액션 유형
  6. Connection Name : 대상 DB 커넥션명
  7. Database Type : 대상 데이터베이스 유형
  8. Privilege Name : 사용자 접속 권한
  9. Client IP : 사용자 클라이언트 IP 주소
  10. Replication Type : DB Replication 유형
  11. DB Host : 접속한 DB 호스트
  12. DB Name : DB명
  13. DB User : DB 사용자 ID
  14. Table(s) : 호출 테이블명
  15. SQL Type : SQL 구문 유형
  16. Query : 실행 쿼리문
  17. Client Name : 이용 클라이언트명 (DataGrip 등)
  18. Execution Reason : 쿼리 실행 사유
  19. Executed From : 쿼리 실행 주체
