## Overview
QueryPie에 등록된 서버의 접속 계정 패스워드를 주기적으로 변경 Job을 등록합니다. 서버 그룹 단위로 Job 을 생성하며, 패스워드 변경은 서버 그룹에 속한 개별 서버를 대상으로 계정별 프로비저닝을 수행합니다.

QueryPie를 통한 서버 계정의 패스워드 변경을 위해선, 각 서버에 패스워드 변경 권한이 있는 계정을 등록해야 합니다. 또한 하나의 Job을 통해 여러 서버의 계정 패스워드를 변경하기 위해선, 각 서버에 같은 계정명과 패스워드로 Job에서 사용할 서버 계정을 생성해야 합니다.

## 패스워드 변경 Job 생성하기 
Administrator > Servers > Server Account Management > Password Provisioning > Create Job

1. Administrator > Servers > Server Account Management > Password Provisioning 메뉴로 이동합니다
2. 우측 상단의 + Create Job 버튼을 클릭합니다.
3. Job 생성을 위한 다음의 정보들을 입력합니다.Name : 패스워드 변경 Job의 Name을 입력합니다.Provisioning Information : 실제 서버에 접속해 패스워드 변경을 수행하는 크리덴셜 정보를 입력합니다.Account : 실제 서버에 접속해 패스워드 변경을 수행하는 account를 입력합니다.Password : 실제 서버에 접속해 패스워드 변경을 수행하는 계정의 Password를 입력합니다.Job Frequency : 패스워드 변경 Job의 반복 주기를 설정합니다. 수동 변경 또는 주기적인 변경을 설정할 수 있습니다.Provisioning Target : 패스워드를 변경할 서버가 그룹핑된 Server Group를 선택합니다. Server Group의 각 개별 서버에 접속하여, 각 계정의 패스워드를 변경합니다. 서버 계정의 패스워드를 주기적으로 변경하기 위해선,  Server Groups의 Accounts 항목의 “Provisioning” 항목이 On으로 되어있어야 합니다.
  1. Name : 패스워드 변경 Job의 Name을 입력합니다.
  2. Provisioning Information : 실제 서버에 접속해 패스워드 변경을 수행하는 크리덴셜 정보를 입력합니다.Account : 실제 서버에 접속해 패스워드 변경을 수행하는 account를 입력합니다.Password : 실제 서버에 접속해 패스워드 변경을 수행하는 계정의 Password를 입력합니다.
    1. Account : 실제 서버에 접속해 패스워드 변경을 수행하는 account를 입력합니다.
    2. Password : 실제 서버에 접속해 패스워드 변경을 수행하는 계정의 Password를 입력합니다.
  3. Job Frequency : 패스워드 변경 Job의 반복 주기를 설정합니다. 수동 변경 또는 주기적인 변경을 설정할 수 있습니다.
  4. Provisioning Target : 패스워드를 변경할 서버가 그룹핑된 Server Group를 선택합니다. Server Group의 각 개별 서버에 접속하여, 각 계정의 패스워드를 변경합니다. 서버 계정의 패스워드를 주기적으로 변경하기 위해선,  Server Groups의 Accounts 항목의 “Provisioning” 항목이 On으로 되어있어야 합니다.
4. Save 버튼을 통해 저장합니다.

Provisioning Information 계정의 패스워드를 주기적으로 변경하기 위해선, Server Group에 계정이 등록되어야 합니다.
