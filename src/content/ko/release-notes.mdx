# Release Notes

<Callout type="info">
QueryPie 공식 릴리스 정보를 버전별로 확인할 수 있습니다.
</Callout>

### Release Notes

*  **New Features** : 해당 버전에 신규로 제공하는 기능을 소개합니다.
*  **Enhancement**  : 기존에 제공하던 기능의 성능이 향상되거나 사용성과 편의성이 개선된 경우 이 항목에 소개합니다.
*  **Bug Fix**  : 고객사로부터 접수된 문제나 결함이 해결된 경우 이 항목에 표기합니다.
*  **External API**  : 신규로 제공되거나 기존 API의 변경 사항을 안내합니다.
