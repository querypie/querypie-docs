다음은 WAC을 처음 접하시는 분들이 자주 궁금해 하시는 질문들입니다.

혹시 더 궁금한 내용이 있으시다면 QueryPie Support를 통해 문의해주세요.


### 익스텐션 업데이트 안내 모달이 떴는데, 어떤 경우에 업데이트가 필요한가요?

WAC 기능 업데이트에 따라서 익스텐션 업데이트가 필수적인 경우에 업데이트 안내 모달이 노출되고 있어요.

현재 사용중인 익스텐션 버전과, QueryPie Host 버전에 따른 최소 요구 버전을 비교하여, 현재 사용중인 버전이 최소 요구 버전보다 낮다면 업데이트를 반드시 진행하셔야 합니다.


### 레코딩 안내 모달은 언제 뜨는건가요? 맨날 뜨는 것 같아요.

레코딩 안내 모달은 세션 시작 시 노출됩니다.

세션의 기준은, 웹앱에 대한 최초의 탭이 뜰 때 시작되고, 모든 탭이 마지막으로 닫혔을때 끝납니다.


### 스크롤 업, 다운을 했는데 기록이 안남아요. 언제 남는건가요?

웹 앱에서 User Activity Recording이 켜져 있고, 스크롤 업, 스크롤 다운 이벤트를 캡쳐하도록 설정되어 있어야 합니다.

그리고 웹 브라우저에서 500px 이상 스크롤을 올리거나 내렸을 경우 기록이 됩니다.

이벤트 기록에는 일정 시간의 디바운싱이 걸려 있어, 여러 종류의 이벤트와 빠른 시간 내에 교차되면 기록되지 않을 수 있습니다.


### 웹 앱 내의 링크를 “새 탭으로 열기” 했더니 Event Snapshot이 남지 않아요.

새 탭으로 링크를 여는 경우, 새로 연 탭에서 불러온 화면을 Extension에서 알 수 없어, Navigated 이벤트 상의 스크린샷 캡쳐가 되지 않습니다. :joy:

다만 다른 이벤트 (마우스 클릭 등) 가 발생했다면 스크린샷이 바로 남는 점 참고 부탁드려요.

익스텐션의 로그인이 풀려있어서 발생하는 현상입니다.

웹브라우저에서 익스텐션 상 Host로 등록한 QueryPie에 접속 후 로그인 하면, 익스텐션에서도 로그인 정보를 불러옵니다. 이후 Role을 선택하면 정상적으로 사용 가능합니다.
