# Overview

QueryPie에서는 Web Apps 자산에 대한 조회, 생성, 수정 사례를 지원합니다. 웹 애플리케이션을 통해 등기된 웹앱, API 또는 수동으로 등록한 웹앱 모두 동일한 페이지에서 관리합니다.

## Web App 조회하기

1. Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps 메뉴로 이동합니다.
2. 테이블 좌측 상단의 검색창을 통해 웹앱명을 조건으로 검색이 가능합니다.
3. 테이블 우측 상단의 새로고침 버튼을 통해 웹앱 목록을 최신화할 수 있습니다.
4. 테이블에서 아래의 컬럼 정보를 제공합니다:
    1. Name : 웹앱명
    2. URL : 웹앱 URL 또는 도메인
    3. Description : 웹앱 설명
    4. Tags : 웹앱에 달린 태그 목록
    5. User Activity Recording : 사용자 활동 기록 여부 (웹앱 접근 및 사용 기록 추적 설정)
    6. Created At : 웹앱 생성일시
    7. Updated At : 웹앱 마지막 수정일시
5. 각 행을 클릭하면 웹앱 세부 정보 조회가 가능합니다.

## Web App 생성하기

1. Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps 메뉴로 이동합니다.
2. 우측 상단의 + Create Web App 버튼을 클릭합니다.

## Web App 삭제하기

1. Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps 메뉴로 이동합니다.
2. 테이블 내 삭제할 대상 웹앱 좌측의 체크 박스를 체크합니다.
3. 테이블 컬럼 라인에 나타난 Delete 버튼을 클릭합니다.
4. 팝업이 나타나면 DELETE 문구를 기입하고 Delete 버튼을 클릭하여 삭제를 진행합니다.

<Callout type="important">
Web App 삭제 시 사용자/그룹에 부여된 해당 웹앱에 대한 접근 권한 모두 일괄 삭제됩니다.
</Callout>
