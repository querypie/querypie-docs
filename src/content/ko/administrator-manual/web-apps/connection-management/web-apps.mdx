# Web Apps

## Overview

QueryPie에서는 Web Apps 자산에 대한 조회, 생성, 수정 사례를 지원합니다. 웹 애플리케이션을 통해 등기된 웹앱, API 또는 수동으로 등록한 웹앱 모두 동일한 페이지에서 관리합니다.

### Web App 조회하기

1. Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps 메뉴로 이동합니다.
2. 테이블 좌측 상단의 검색창을 통해 웹앱명을 조건으로 검색이 가능합니다.
3. 테이블 우측 상단의 새로고침 버튼을 통해 웹앱 목록을 최신화할 수 있습니다.
4. 테이블에서 아래의 컬럼 정보를 제공합니다:
    1. Name : 웹앱명
    2. URL : 웹앱 URL 또는 도메인
    3. Description : 웹앱 설명
    4. Tags : 웹앱에 달린 태그 목록
    5. User Activity Recording : 사용자 활동 기록 여부 (웹앱 접근 및 사용 기록 추적 설정)
    6. Created At : 웹앱 생성일시
    7. Updated At : 웹앱 마지막 수정일시
5. 각 행을 클릭하면 웹앱 세부 정보 조회가 가능합니다.

### Web App 생성하기

<figure data-layout="center" data-align="center">
![Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps](/administrator-manual/web-apps/connection-management/web-apps/Screenshot-2025-08-27-at-6.58.08-PM.png)
<figcaption>
Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps
</figcaption>
</figure>

1. Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps 메뉴로 이동합니다.
2. 우측 상단의 + Create Web App 버튼을 클릭합니다.
3. Web App 생성화면
    1. Name : 웹앱명
    2. Base URL : 웹앱 URL 또는 호스트 주소 
    3. Description : 웹앱 설명
    4. Tags : 웹앱에 추가할 태그 설정
    5. Watermark : 브라우저에 워터마크 적용 여부
    6. User Activity Recording : 사용자 활동 기록 여부 (웹앱 접근 및 사용 기록 추적 설정)
    7. URL Paths: Web App의 하위 경로 입력

### Web App Owner 및 Member 설정하기

<figure data-layout="center" data-align="center">
![Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps &gt; Web App Owners](/administrator-manual/web-apps/connection-management/web-apps/Screenshot-2025-08-27-at-6.57.07-PM.png)
<figcaption>
Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps &gt; Web App Owners
</figcaption>
</figure>

Web App에 Owner와 Member를 지정하여 Just-In-Time (JIT) 접근 요청에 대한 권한을 세분화할 수 있습니다. 각 역할의 권한은 다음과 같이 명확하게 분리됩니다.

*  **Web App Owner:** JIT 접근 요청에 대한 승인 권한을 가집니다. 
*  **Web App Member:** JIT 접근 요청에 대한 상신(신청) 권한을 가집니다. 
* 한 명의 사용자가 Owner와 Member 권한을 동시에 가질 수 있으며, 이 경우 요청 상신과 승인이 모두 가능합니다.

<figure data-layout="center" data-align="center">
![Assign Owners 화면](/administrator-manual/web-apps/connection-management/web-apps/Screenshot-2025-08-27-at-6.57.24-PM.png)
<figcaption>
Assign Owners 화면
</figcaption>
</figure>

Web App 생성 시 또는 상세 정보 화면에서 Owner/Member를 설정할 수 있습니다. `+ Assign Owners` 또는 `+ Assign Members` 버튼을 클릭하면 사용자 추가 모달이 나타납니다.

모달 창에서는 Status가 Active인 사용자만 조회되며, 원하는 사용자를 선택하여 추가할 수 있습니다.

### Web App 삭제하기


1. Administrator &gt; Web Apps &gt; Connection Management &gt; Web Apps 메뉴로 이동합니다.
2. 테이블 내 삭제할 대상 웹앱 좌측의 체크 박스를 체크합니다.
3. 테이블 컬럼 라인에 나타난 Delete 버튼을 클릭합니다.
4. 팝업이 나타나면 DELETE 문구를 기입하고 Delete 버튼을 클릭하여 삭제를 진행합니다.

<Callout type="important">
Web App 삭제 시 사용자/그룹에 부여된 해당 웹앱에 대한 접근 권한 모두 일괄 삭제됩니다.
</Callout>
