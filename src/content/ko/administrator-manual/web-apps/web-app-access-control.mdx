## Overview

해당 문서에서는 QueryPie에서 웹앱 접근 권한을 효과적으로 관리하기 위한 가이드를 확인할 수 있습니다. RBAC을 통해 사전에 정의한 접근 권한을 반복하여 부여할 수 있는 방법을 제공하고 있으며, URL 기반의 웹앱 접근 제어를 통해 사용자별로 허용된 웹 애플리케이션에만 접근할 수 있도록 관리합니다. 또한 정책 관리 기능을 통해 세부적인 접근 규칙을 설정할 수 있습니다.
