## Overview

패스워드를 주기적으로 변경하는 Job 을 생성하고 관리하는 메뉴입니다. 서버 그룹 단위로 Job 을 생성하며, 패스워드 변경은 서버 그룹에 속한 개별 서버를 대상으로 계정별 프로비저닝을 수행합니다.

<Callout type="info">
이 메뉴는 Administrator &gt; General &gt; Company Management &gt; [Security](https://chequer.atlassian.net/wiki/spaces/QM/pages/544178422/Security#%EC%84%9C%EB%B2%84-%EC%BB%A4%EB%84%A5%EC%85%98-%EB%B3%B4%EC%95%88-%EC%84%A4%EC%A0%95)에서 Server Connection Security의  **Password Provisioning**  설정을 On으로 선택 후 저장해야 표시됩니다.
</Callout>


## 패스워드 변경 Job 조회하기

<figure data-layout="center" data-align="center">
![Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning](/administrator-manual/servers/server-account-management/password-provisioning/Screenshot-2025-07-27-at-6.31.29-PM.png)
<figcaption>
Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning
</figcaption>
</figure>

1. Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning 메뉴로 이동합니다.
2. 테이블 좌측 상단의 검색란을 통해 Job의 Name 또는 Provisioning Account의 검색이 가능합니다.
3. 테이블 우측 상단의 새로고침 버튼을 통해 계정 목록을 최신화할 수 있습니다.
4. 테이블에서 이하의 컬럼 정보를 제공합니다.
    1.  **Name** : 패스워드 변경 Job의 Name
    2.  **Provisioning Account**  : 실제 서버에 접속해 패스워드 변경을 수행하는 Account
    3.  **Job Frequency**  : Job 실행 주기
    4.  **Latest Result**  : 마지막 실행 결과
    5.  **Last Run At**  : 마지막 실행 일시
5. 각 행을 클릭하면 Job의 세부 정보 조회가 가능합니다.

## 패스워드 변경 Job 생성하기

1. Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning 메뉴로 이동합니다.
2. 우측 상단의 `+ Create Job` 버튼을 클릭합니다.
3. [패스워드 변경 Job 생성하기](패스워드 변경 Job 생성하기) 가이드를 참고하여 서버를 등록합니다.


## 패스워드 변경 Job 삭제하기

<figure data-layout="center" data-align="center">
![image-20240831-161802.png](/administrator-manual/servers/server-account-management/password-provisioning/image-20240831-161802.png)
</figure>

1. Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning 메뉴로 이동합니다.
2. 테이블 내 삭제할 대상 Job 좌측의 체크 박스를 체크합니다.
3. 테이블 컬럼 라인에 나타난 `Delete` 버튼을 클릭합니다.
4. 팝업이 나타나면 `OK` 버튼을 클릭하여 삭제를 진행합니다.
