## Overview

커넥션에 접속할 수 있는 계정을 미리 등록하여 템플릿 형태로 저장해 둘 수 있습니다. 이렇게 미리 생성한 템플릿은 서버 또는 서버 그룹에서 쉽고 편리하게 불러올 수 있습니다.


## 서버 계정 템플릿 생성하기

<figure data-layout="center" data-align="center">
![Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templates &gt; Create Template](/administrator-manual/servers/server-account-management/server-account-templates/image-20240828-011910.png)
<figcaption>
Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templates &gt; Create Template
</figcaption>
</figure>

1. Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templates 메뉴로 이동합니다.
2. 우측 상단의 `Create Template` 버튼을 클릭합니다. 
3. 템플릿 생성을 위한 다음의 정보들을 입력합니다. 
    1.  **Name** : 템플릿을 구분하기 위한 이름을 입력합니다. 
    2.  **Description** : 템플릿을 설명할 수 있는 내용을 입력합니다. 
4. `Save` 버튼을 통해 저장합니다. 
5. 목록에서 새롭게 생성된 템플릿을 클릭합니다. 
6. Account 항목에서 `Add Account`를 클릭하여 템플릿에 등록할 계정을 추가합니다. 
    * 개별 Account 설정 방법은 [서버를 그룹으로 관리하기](서버를 그룹으로 관리하기)를 참고합니다. 
7. 우측 상단의 `Save Changes` 를 클릭하여 저장합니다.
