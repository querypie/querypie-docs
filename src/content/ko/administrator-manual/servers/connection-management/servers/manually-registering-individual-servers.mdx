## Overview
QueryPie에서는 접근 제어를 적용할 서버를 수동으로 등록할 수 있습니다. 등록된 개별 서버는 서버 그룹에 추가하여 접근 권한을 부여하거나 정책 설정을 할 수 있습니다. 단, Windows 서버의 경우 수동 등록이 불가능하며, Server Agent 연동을 통한 자동 등록만 가능합니다.



## 수동으로 서버 등록하기
개별 서버를 수동으로 등록하기 위해서는 서버의 기본적인 정보 입력이 필요합니다.

<p align="center">
<div>[image-20240828-154238.png]()</div>
*Administrator &gt; Servers &gt; Connection Management &gt; Servers &gt; Create Server*
</p>

### 1. Information 입력하기
<p align="center">
<div>[image-20240828-154305.png]()</div>
</p>

1. Administrator &gt; Servers &gt; Connection Management &gt; Servers 메뉴로 이동합니다.
2. 우측 상단의 `Create Server` 버튼을 클릭합니다.
3. 서버 등록을 위한 다음의 정보들을 입력합니다.
    1. **Name**  : 해당 서버를 구분할 수 있는 이름을 입력합니다.
    2. **Host**  : 서버의 호스트를 입력합니다. 도메인, IP 형태 모두 입력할 수 있습니다.
    3. **Server OS**  : 서버의 OS를 선택합니다.
    4. **OS Version ** : 서버 OS의 상세 정보를 기입합니다.
    5. **Port**  : 커넥션의 포트를 입력합니다.
        1. SSH/SFTP : SSH/SFTP 접속시 사용할 포트를 입력합니다.
        2. TELNET : TELNET 접속시 사용할 포트를 입력합니다. [Security](https://chequer.atlassian.net/wiki/spaces/QM/pages/544178422/Security#%EC%84%9C%EB%B2%84-%EC%BB%A4%EB%84%A5%EC%85%98-%EB%B3%B4%EC%95%88-%EC%84%A4%EC%A0%95) 메뉴 “Using insecure protocols“ 항목을 통해 TELNET을 활성화한 경우에만 표시됩니다.
        3. FTP : FTP 접속시 사용할 포트를 입력합니다. [Security](https://chequer.atlassian.net/wiki/spaces/QM/pages/544178422/Security#%EC%84%9C%EB%B2%84-%EC%BB%A4%EB%84%A5%EC%85%98-%EB%B3%B4%EC%95%88-%EC%84%A4%EC%A0%95) 메뉴 “Using insecure protocols“ 항목을 통해 FTP를 활성화한 경우에만 표시됩니다.

### 2. Tag 입력하기 (Optional)
<p align="center">
<div>[image-20240723-163036.png]()</div>
</p>

필요시 개별 서버에 Tag를 수동으로 입력할 수 있습니다. 이렇게 추가된 Tag는 서버 목록에서 또는 서버 그룹에 서버를 추가할 시 해당 Tag를 활용하여 필터링할 수 있습니다. 또한 Cloud Provider 동기화 해온 태그도 함께 표시됩니다. (단, 동기화를 통해 불러온 태그는 삭제 및 수정 불가)

1. **Key**  : 태그를 구분할 수 있는 Key 값을 512자 이내로 입력합니다. Key 값을 필수로 입력해야 하며, 이미 등록된 키는 중복 입력이 불가합니다. 중복은 대소문자를 구분하여 체크합니다.
2. **Value**  : 필터링에 사용할 Value 값을 256자 이내로 입력합니다.

위 과정을 거친 후 최종 `Save` 버튼을 클릭하면 서버가 성공적으로 등록됩니다.
