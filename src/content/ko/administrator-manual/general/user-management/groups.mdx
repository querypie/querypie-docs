# Groups

### Overview

QueryPie에서는 사용자를 그룹 단위로 묶어 일괄 관리할 수 있는 그룹 기능을 제공합니다. 그룹은 DB 접근 권한, Workflow 승인/검토자 역할, 서버나 쿠버네티스 역할(Role) 부여 시의 단위로 사용될 수 있습니다. Groups 페이지에서는 QueryPie에 등록된 사용자 그룹 목록을 조회 및 관리합니다.


### 그룹 목록 조회하기

Administrator &gt; General &gt; User Management &gt; Groups 페이지로 진입하면 그룹 목록을 조회할 수 있습니다. 그룹 이름으로 검색할 수 있습니다. 조회 가능한 정보는 다음과 같습니다.

<figure data-layout="center" data-align="center">
![Administrator &gt; General &gt; User Management &gt; Groups](/administrator-manual/general/user-management/groups/image-20240714-053938.png)
<figcaption>
Administrator &gt; General &gt; User Management &gt; Groups
</figcaption>
</figure>

*  **Auth Provider**  : 사용자 인증 제공자
    * QueryPie 또는 User Management &gt; Authentication 에서 설정된 IdP Type을 표시합니다.
    * 수동으로 등록된 그룹의 경우 QueryPie로 표시됩니다.
    * IdP에서 연동된 그룹은 각 ID Provider 이름으로 표시됩니다.
*  **Group Name** : 그룹 이름
*  **Members**  : 그룹에 속한 사용자 목록
*  **Updated At** : 그룹 정보가 마지막으로 변경된 일시
*  **Created At** : 그룹 생성 일시


### 그룹 수동 추가하기

Groups 페이지 우상단 `Create Group` 버튼을 클릭하면 모달이 출력되며, 원하는 이름을 입력하고 `Save` 버튼을 클릭하면 그룹 생성이 완료됩니다.

<Callout type="important">
수동 추가된 그룹에 대해서만 사용자 추가 또는 삭제가 가능합니다.<br/>IdP에서 동기화된 그룹에 대해서는 조회만 지원하며, 사용자 목록 및 그룹 관리는 IdP 에서 수행하여야 합니다.
</Callout>


### 그룹에 사용자 추가하기

리스트에서 사용자를 추가하려는 그룹을 클릭하면 Drawer가 열리고, 상세 정보를 확인할 수 있습니다.

<figure data-layout="center" data-align="center">
![Administrator &gt; General &gt; User Management &gt; Groups &gt; 상세 정보](/administrator-manual/general/user-management/groups/screenshot-20240726-121151.png)
<figcaption>
Administrator &gt; General &gt; User Management &gt; Groups &gt; 상세 정보
</figcaption>
</figure>

1. `+ Add Users` 버튼을 클릭하면 모달이 출력됩니다.
    1. 이름 또는 이메일로 사용자를 검색할 수 있습니다.
    2. 11.2.0부터 LDAP을 통해 동기화된 비활성화 상태의 사용자도 검색 결과에 포함되어 표시됩니다. 비활성 사용자는 이름 옆에 `(Inactive)` 텍스트가 표시됩니다.
    3. 추가하려는 사용자를 클릭하면 추가 상태로 변경됩니다.
    4. 하단 `Check all` 버튼을 클릭하면 전체 사용자가 선택됩니다.<br/> 
2. `Save` 버튼을 클릭하여 그룹 변경 내역을 저장합니다. 그룹 리스트 및 상세 정보 Drawer에서 추가된 사용자를 확인할 수 있습니다.

<Callout type="info">
**Q. 그룹에 특정 사용자를 추가하려고 하는데 모달에서 보이지 않아요.**
A. 사용자 추가 모달에는 활성화 상태의 사용자만 표시됩니다. Users 페이지에서 사용자의 계정 상태가 Inactive 인지 확인해보세요.
</Callout>


### 그룹에서 사용자 제거하기

1. 그룹 상세 페이지에서 제거하려는 사용자를 체크박스로 선택한 뒤, 테이블 헤더에 표시되는 `Delete` 버튼을 클릭합니다.
2. 사용자 제거 확인 모달에서 `OK` 버튼을 클릭합니다.
3. 사용자 제거가 완료되면 Drawer 내 사용자 리스트 및 Group 리스트의 목록에서 확인할 수 있습니다.

Administrator &gt; General &gt; User Management &gt; Groups &gt; 상세 정보

### 그룹 이름 변경하기

그룹 상세 정보 Drawer 최상단의 `Edit` 버튼을 클릭하면, 수정 모달이 출력됩니다. 그룹 이름을 변경 후 `Save` 버튼을 클릭하여 저장합니다.

<figure data-layout="center" data-align="center">
![Administrator &gt; General &gt; User Management &gt; Groups &gt; 상세 정보 &gt; Update Group](/administrator-manual/general/user-management/groups/screenshot-20240726-122036.png)
<figcaption>
Administrator &gt; General &gt; User Management &gt; Groups &gt; 상세 정보 &gt; Update Group
</figcaption>
</figure>


### 그룹 삭제하기

Groups 리스트에서 삭제하려는 그룹을 체크박스로 선택합니다. 테이블 헤더에 `Delete` 버튼이 표시됩니다. 버튼을 클릭하고, 확인 모달에서 `OK` 버튼을 클릭하여 삭제를 완료합니다.

<figure data-layout="center" data-align="center">
![screenshot-20240726-123314.png](/administrator-manual/general/user-management/groups/screenshot-20240726-123314.png)
</figure>

<Callout type="info">
그룹 이름 변경 및 삭제는 QueryPie에서 수동으로 추가된 그룹만 가능합니다.
</Callout>

<Callout type="important">
그룹이 삭제되는 경우 그룹에 속한 사용자가 삭제되지는 않지만, 그룹을 매개로 부여받은 권한 및 역할은 즉시 해제됩니다. 그룹 삭제는 취소가 불가능하므로 실행에 신중을 기하시기 바랍니다.
</Callout>
