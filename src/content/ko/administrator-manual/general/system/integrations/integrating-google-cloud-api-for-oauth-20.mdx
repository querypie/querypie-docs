## Overview
Google Cloud API에 생성한 Client ID 및 Client Secret을 정보를 등록하여 BigQuery 접속에 OAuth 2.0 방식의 인증을 통합할 수 있습니다. Google Cloud API 콘솔에서 OAuth 2.0 을 설정하는 방법은 이 문서를 참고 부탁드립니다.

### OAuth 2.0 Client ID 추가
1. General &gt; System &gt; Integration 메뉴를 선택합니다.
2. Authentication 항목 아래에서 Google Cloud API 타일을 클릭합니다.
3. OAuth 2.0 Client IDs 항목의 `+Add` 버튼을 클릭합니다.
4. 팝업 창에 정보를 입력하고 Save 버튼을 눌러 저장합니다.
    1. Name : 목록에서 식별하기 쉬운 이름을 입력합니다.
    2. Client ID : Google API Console에서 확인한 Client ID 를 입력합니다.
    3. Client Secret : Google API Console에서 확인한 Client ID 를 입력합니다.

### OAuth 2.0 Client ID 수정.
1. 목록에서 수정하고자하는 특정 Client ID 행을 클릭합니다.
2. 팝업창에 정보를 수정하고 Save 버튼을 눌러 저장합니다.

### OAuth 2.0 Client ID 삭제
1. 목록에서 삭제를 원하는 특정 Client ID를 체크합니다.
2. 상단의 Delete 버튼을 눌러 삭제합니다.  이 때 삭제하고자 하는 Client ID가 커넥션 설정과 연결되어 있으면 다음과 같은 경고 메시지가 출력되고 삭제할 수 없습니다. 먼저 커넥션 설정에서 OAuth 설정을 제거하거나 커넥션을 삭제해야 합니다.
