## Overview
QueryPie의 System Access Control에서 발생하는 로그들을 Server Logs에서 확인할 수 있습니다. 서버 접속 권한 부여와 회수, 서버 로그인/로그아웃, 수행한 명령어 이력 등의 정보를 확인할 수 있습니다. 그 외, QueryPie를 통해 접속한 세션 리스트을 조회하거나 세션을 강제종료 할 수 있습니다.

## 연관 감사 로그 종류
쿼리파이 공통으로 제공되는 감사 로그에 더하여 쿼리파이 시스템 접근제어 (QueryPie SAC) 라이선스 보유 시, 이하의 감사 로그의 종류 제공합니다.

| **공통 로그**                                                          | **SAC 전용 로그**                                                                                                                              |
| ------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------ |
| **General** [ul]User Access HistoryActivity LogsAdmin Role History | **Servers** [ul]Server Access HistoryCommand AuditSession LogsSession MonitoringAccess Control LogsServer Role HistoryAccount Lock History |

1. **General**

* User Access History : 사용자 로그인, 로그아웃 이력
* Activity Logs : 관리자가 수행하는 리소스 등록 및 설정 변경 이력
* Admin Role History : 관리자 권한 부여, 변경, 회수 이력

1. **Servers**

* [Server Access History](Server Access History) : Server 접속 이력
* [Command Audit](Command Audit) : Server내 수행 명령어 이력
* [Session Logs](Session Logs) : 연결 세션 내 행위 녹화 내역
* [Session Monitoring](Session Monitoring) : 접속한 세션 리스트 조회
* [Access Control Logs](Access Control Logs) : Direct Permission 부여 이역
* [Server Role History](Server Role History) : Role 부여 이력
* [Account Lock History](Account Lock History) : QueryPie사용자별 서버 접속 계정 잠금 및 해제 이력
