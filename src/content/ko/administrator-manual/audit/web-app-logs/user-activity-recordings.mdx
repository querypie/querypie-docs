## Overview

조직에서 관리하는 웹 애플리케이션 사용자 활동을 기록합니다.

## User Activity Recordings 조회하기

<figure data-layout="center" data-align="center">
![Administrator &gt; Audit &gt; Web Apps &gt; User Activity Recordings](/administrator-manual/audit/web-app-logs/user-activity-recordings/image-20250629-175520.png)
<figcaption>
Administrator &gt; Audit &gt; Web Apps &gt; User Activity Recordings
</figcaption>
</figure>

1. Administrator &gt; Audit &gt; Web Apps &gt; User Activity Recordings 메뉴로 이동합니다.
2. 현재 월 기준으로 로그가 내역갯수로 조회됩니다.
3. 테이블 최측 상단의 검색란을 통해 아래와 조건으로 검색이 가능합니다.
    1.  **User Name**  : 사용자 이름
    2.  **User Email**  : 사용자 Email
    3.  **Role Name**  : 역할명
    4.  **Web App Name**  : 웹 애플리케이션 이름
    5.  **Web App Base Url**  : 웹 애플리케이션 기본 URL
    6.  **Session ID**  : 세션 고유 식별자
    7.  **Client Ip**  : 클라이언트 IP 주소
4. 검색 필드 우측 필터 버튼을 클릭하여 AND 조건으로 아래와 필터링이 가능합니다.

**a. Action At**  : 액션 발생 일시 범위


## User Activity Recordings 상세 내역 조회하기

각 행을 클릭하면 세부 정보 조회가 가능합니다.

<figure data-layout="center" data-align="center">
![Administrator &gt; Audit &gt; Web Apps &gt; User Activity Recordings &gt; User Activity Recordings Details](/administrator-manual/audit/web-app-logs/user-activity-recordings/image-20250629-175637.png)
<figcaption>
Administrator &gt; Audit &gt; Web Apps &gt; User Activity Recordings &gt; User Activity Recordings Details
</figcaption>
</figure>

1.  **상세 페이지에는 아래와 정보를 노출합니다:** 
    1.  **Action At**  : 액션 시점
    2.  **Name**  : 대상 사용자 이름
    3.  **Role**  : 사용자 역할명
    4.  **Client Ip**  : 클라이언트 IP 주소
    5.  **Session ID**  : 세션 고유 식별자
    6.  **Email**  : 대상 사용자 이메일
    7.  **Web App**  : 웹 애플리케이션명
    8.  **Base URL**  : 웹 애플리케이션 기본 URL
2.  **Event Timeline에서는 관리자가 기록하도록 설정한 항목만 표시됩니다** 
    1. Navigated : 페이지 이동 기록 (설정 시)
    2. Mouse Clicked : 마우스 클릭 이벤트 (설정 시)
    3. Clipboard Copied : 클립보드 복사 이벤트 (설정 시)
    4. File Uploaded : 파일 업로드 이벤트 (설정 시)
    5. File Downloaded : 파일 다운로드 이벤트 (설정 시)
    6. Value Changed : 값 변경 이벤트 (설정 시)
    7. Scrolled Up/Down : 스크롤 이벤트 (설정 시)
    8. Print Initiated : 인쇄 시작 이벤트 (설정 시)
    9. URL : 접근한 페이지 URL
    10. Tab ID : 탭 식별자
    11. Timestamp : 이벤트 발생 시간
3.  **화면 우측에는 실제 사용자가 접근한 웹 애플리케이션의 스크린샷이 표시됩니다.** 

<Callout type="info">
관리자는 Web Apps &gt; Web App Configurations에서 각 웹 애플리케이션별로 기록할 사용자 활동 유형을 설정할 수 있습니다.
</Callout>
