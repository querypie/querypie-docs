## Overview
조직에서 관리하는 웹 애플리케이션 이벤트 감사 이력을 기록합니다.



## Web Event Audit 조회하기
<p align="center">
<div>[image-20250629-172952.png]()</div>
*Administrator &gt; Audit &gt; Web Apps &gt; Web Event Audit*
</p>

1. Administrator &gt; Audit &gt; Web Apps &gt; Web Event Audit 메뉴로 이동합니다.
2. 현재 월 기준으로 로그가 내림차순으로 조회됩니다.
3. 테이블 최측 상단의 검색란을 통해 아래와 조건으로 검색이 가능합니다.
    1. User Name : 사용자 이름
    2. User Email : 사용자 이메일
    3. Role Name : 역할명
    4. Web App Name : 웹 애플리케이션 이름
    5. Content : 콘텐츠 내용
    6. Url : 접근한 URL
    7. Session Uuid : 세션 고유 식별자
    8. Tab Id : 탭 식별자
    9. Client Ip : 클라이언트 IP 주소
    10. Message : 메시지 내용
4. 검색 필드 우측 필터 버튼을 클릭하여 AND/OR 조건으로 아래와 필터링이 가능합니다.
    1. Event Type : 이벤트 유형 (Mouse Clicked/Clipboard Copied 등)
    2. Result : 실행 결과 (Success/Failure)
    3. Action At : 액션 발생 일시 범위
5. 테이블 우측 상단의 새로고침 버튼을 통해 로그 목록을 최신화할 수 있습니다.
6. 테이블에서 이하의 컬럼 정보를 제공합니다:
    1. **No**  : 순번
    2. **Action At**  : 액션 발생 일시
    3. **Event Type**  : 이벤트 유형 (Mouse Clicked, Clipboard Copied 등)
    4. **Result**  : 실행 결과 (Success/Failure)
    5. **Name**  : 사용자명
    6. **Email**  : 사용자 이메일
    7. **Web App**  : 웹 애플리케이션명
    8. **URL**  : 접근한 URL
    9. **Session ID**  : 세션 고유 식별자
    10. **Tab ID**  : 탭 식별자
    11. **Client IP**  : 클라이언트 IP 주소

## Web Event Audit 상세 내역 조회하기
각 행을 클릭하면 세부 정보 조회가 가능합니다.

<p align="center">
<div>[image-20250629-172959.png]()</div>
*Administrator &gt; Audit &gt; Web Apps &gt; Web Event Audit &gt; Event Audit Detail*
</p>

* **우측 드로워에는 아래와 정보를 노출합니다:**
    1. **Action At**  : 액션 시점
    2. **Event Type**  : 이벤트 유형 (Mouse Clicked, Clipboard Copied 등)
    3. **Result**  : 이벤트 처리 성공/실패 여부
        1. ✅ Success
        2. ❌ Failure
    4. **Client IP**  : 클라이언트 IP 주소
    5. **Name**  : 대상 사용자 이름
    6. **Email**  : 대상 사용자 이메일
    7. **Role**  : 사용자 역할명
    8. **Message**  : 이벤트 실행 등 특이사항에 대한 기록
    9. **Web App**  : 웹 애플리케이션명
    10. **URL**  : 이벤트가 발생한 페이지 URL
    11. **Session ID**  : 세션 고유 식별자
    12. **Tab ID**  : 탭 식별자
    13. **Clicked Text**  : 클릭된 텍스트 내용 (Mouse Clicked 이벤트의 경우)
    14. **Event Snapshot**  : 이벤트 발생 시점의 화면 스크린샷
