## Overview

QueryPie 관리자 권한의 부여 및 회수 이력을 조회하는 페이지입니다.

Administrator &gt; General &gt; User Management &gt; Users 페이지에서 특정 사용자에게 관리자 권한을 부여 또는 회수하는 경우 Admin Role History에 이력이 남게 됩니다.


## Admin Role History 조회하기

![Administrator &gt; Audit &gt; General &gt; Admin Role History](/administrator-manual/audit/general-logs/admin-role-history/screenshot-20240722-190405.png)
<figcaption>
Administrator &gt; Audit &gt; General &gt; Admin Role History
</figcaption>
</figure>

1. Administrator &gt; Audit &gt; General &gt; Admin Role History 메뉴로 이동합니다.
2. Action At 기준 최신 내역부터 오래된 순으로 관리자 권한의 부여 또는 회수 로그가 조회됩니다.
    1.  **Name**  : Display Name이 표시됩니다.
    2.  **Description**  : 부여된 관리자 권한 상세
    3.  **Action By**  : 관리자 권한의 부여 또는 회수 주체 (자동 회수의 경우  **System**  으로 표시)
3. 지원되는 검색 조건은 다음과 같습니다.
    1.  **Name**  : 사용자 이름 (Display Name)
    2.  **Action Type**  : 이벤트 이름 
        * Admin Role Granted
        * Admin Role Revoked
4. 그 외, 이벤트 발생 시점 ( **Action At** ) 기준으로 로그 필터링이 가능합니다.
