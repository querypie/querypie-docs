## Overview

QueryPie KAC은 AWS EKS나 기타 온프레미스 쿠버네티스 클러스터와 같은 클라우드 인프라를 중앙에서 관리하기 위한 Kubernetes API 보호 솔루션입니다. KAC를 사용하면 관리자는 세밀하게 사용자의 쿠버네티스 리소스 API 접근 제한 및 관리를 수행하고, 사용자가 요청한 Kubernetes API를 감사 및 모니터링할 수 있습니다.


## 권장하는 KAC 초기 설정 순서
