## Overview

해당 문서에서는 QueryPie 에서 데이터베이스를 효과적으로 관리하기 위한 가이드를 확인할 수 있습니다. 클라우드 동기화 기능을 통해 Cloud Provider로부터 데이터베이스를 한 번에 동기화 해올 수 있는 방법과 수동으로 데이터베이스를 등록하는 방법을 확인할 수 있습니다.

## 지원 기능

*  **접근제어 및 로깅** : 사용자 또는 그룹의 데이터베이스 접근권한 제어 및 실행된 쿼리 등의 감사로그를 확인할 수 있습니다.
*  **구문제어** : SQL 구문별 명령어 제어 및 데이터 내보내기, 가져오기, 클립보드 복사 기능 등 DB 작업과 관련된 권한을 제어할 수 있습니다.
*  **데이터 정책** : 데이터 소스별로 테이블/컬럼 조회 통제, 마스킹, 민감 정보 알림 정책을 설정하여 최소한의 데이터만 접근하도록 관리합니다.
*  **Proxy 접속** : 외부 클라이언트를 통한 데이터베이스 접속 시 Agent 및 Agentless 방식으로 Proxy 접속을 지원합니다.
*  **DML 전후데이터** : INSERT/UPDATE/DELETE 쿼리 실행 시 변경 전후 데이터를 감사로그로 기록합니다.
*  **원장 테이블 정책** : 중요 테이블을 원장으로 관리하고, DML 쿼리 실행 시 승인 프로세스 및 조회사유 등을 강제할 수 있습니다.
