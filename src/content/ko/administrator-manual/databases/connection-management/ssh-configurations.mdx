## Overview
SSH Configurations는 DB 커넥션 연결시 사용할 SSH 터널링 정보를 관리하는 페이지입니다.

<p align="center">
<div>[스크린샷 2024-07-30 오후 11.02.32.png]()</div>
*Administrator &gt; Databases &gt; Connection Management &gt; SSH Configuration &gt; Create SSH Configuration*
</p>



1. Databases 설정 메뉴에서 Connection Management &gt; SSH Configuration 메뉴로 이동합니다.
2. 우측 상단의 `Create SSH` 버튼을 클릭합니다.
3. SSH 터널링 등록을 위한 정보를 입력합니다.
    1. Name: SSH 터널링 구성 이름
    2. SSH Auth Type: 인증 방식 (Password 또는 Key pair 방식)
        * 선택지에 따라 모달 하단 인증정보 입력 내용이 상이함
    3. SSH Host: SSH 접속을 위한 Host 정보 (도메인, IP 형태 모두 입력 가능)
    4. SSH Port: SSH 접속을 위한 포트 정보
    5. Username / Password : 접속시 사용할 ID 및 비밀번호
        * Password 방식 선택시 표시
    6. SSH Private Key / SSH Key Passphrase : 접속시 사용할 사설키 및 passphrase 정보
        * Key pair 방식 선택시 표시
4. `Save` 버튼을 누르면 입력 내용이 저장됩니다.
