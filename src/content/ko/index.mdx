# QueryPie 제품 문서에 오신 것을 환영합니다

## QueryPie란?

QueryPie는 **DAC, SAC, KAC, WAC**를 하나의 통합 플랫폼에서 제공하는 통합 보안 솔루션입니다. 클라우드와 온프레미스 환경 모두를 아우르며, 데이터베이스부터 시스템, 쿠버네티스, 웹 애플리케이션까지 모든 접근을 통합 관리합니다.

RBAC/ABAC 기반의 동적 접근 제어와 웹 SQL 에디터, 실시간 감사 및 이상 징후 알림을 통해 기업의 데이터 보안을 강화합니다. Docker 기반의 쉬운 설치와 웹 기반 인터페이스로 SaaS의 편리함과 온프레미스의 강력한 보안을 결합한 하이브리드 솔루션을 제공합니다.

## Community Edition 🎉

**QueryPie Community 버전**이 출시되었습니다! Enterprise 버전의 모든 기능을 무료로 체험해보세요.

DAC, SAC, KAC, WAC의 모든 접근 제어 기능을 포함하며, 최대 5명의 사용자 계정으로 운영 가능합니다.
Docker 기반의 간단한 설치로 7-10분 내에 바로 시작할 수 있습니다.
한 줄 명령어 `bash <(curl https://dl.querypie.com/setup.v2.sh)`로 자동 설치가 가능하며, 1년간 유효한 무료 라이선스를 제공합니다.
유효기간이 만료된 시기에, Community Edition 라이선스를 무료로 연장하여 사용할 수 있습니다.

* **[QueryPie 커뮤니티 버전 설치 가이드](https://www.querypie.com/ko/resources/learn/documentation/querypie-install-guide)** - 무료 설치 및 체험 가이드
