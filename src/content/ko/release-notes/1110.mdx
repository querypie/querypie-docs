## QueryPie 11.1.0 Release

2025년 07월 25일


**New Feature**

* [General] 멀티에이전트에서 Server Privilege Request 요청 기능 추가
* [General] Google Analytics 쿠키 사용 동의 및 쿠키 설정 기능 제공
* [General] 워크플로우 승인완료되었거나 Urgent mode에서 Executor가 취소할 수 있는 External API
* [DAC] 신정책 (태그 기반 정책)에서 User scope 로 사용자 속성을 지정하는 기능 추가
* [DAC] 신정책(태그 기반 정책)에서 Policy Audit Log에 Event Type 컬럼 추가
* [DAC] 신정책(태그 기반 정책)에서 Audit Log에 Policy Exception Log 추가
* [DAC] 태그 기반 정책 Masking 정책 걸리면 alert 발송
* [DAC] 태그 기반 정책 Workflow DB Policy Exception Request에 대한 alert 발송
* [DAC] 태그 기반 정책 Sensitive data 정책/Level 걸리면(sensitive data 접근시) alert 발송
* [DAC] 태그 기반 정책 Data Access 정책 걸리면(Restricted data 접근시) alert 발송
* [SAC] QueryPie와 Windows Active Directory 연동하는 기능 추가
* [SAC] Server Privilege Request 신청 시, Command Template 선택 기능 추가
* [SAC] Require Privilege 옵션을 통한 서버 계정 접근 제어 도입
* [SAC] AD 계정 패스워드 변경 기능 추가
* [SAC] RDP Server Agent 원격 업데이트 기능 추가
* [SAC] 사용자의 Role / Direct Permission 만료 예정 목록 API 제공
* [WAC] 관리자가 사용자의 JIT Web 세션에 대한 권한 회수 기능 제공

**Improvement**

* [General] Enable Attribute Synchronization 옵션을 사용하면서 Static IP를 수정할 수 있도록 개선
* [General] 결재자 그룹의 멤버가 비활성되더라도 결재자 지위를 유지하도록 개선 
* [General] 워크플로우 승인 완료 전에도 참조자가 상신 내용 조회할 수 있도록 개선
* [General] 사용자 대시보드 SAC 및 KAC 메뉴에서 Open Agent 클릭 시, Multi-Agent가 열리도록 개선 
* [DAC] 신정책 (태그 기반 정책)에서 정책예외 생성시 Data Scope를 직접 지정하도록 변경 
* [DAC] Database 비밀번호 만료시 Web Editor에서 사용자가 자신의 비밀번호 변경할 수 있도록 개선 (Oracle 19c 한정)
* [DAC] SAP HANA DB의 데이터 유형이 SECONDDATE 로 지정된 값에서 조회시 오류 개선 
* [SAC] 사용자 대시보드에서 Severs에서 서버 접속 가능한 allowed zone 표시
* [SAC] JIT 임시계정 기능 - Global Option 제공
* [SAC] Server Group의 Account UI 개선
* [SAC] One Time Account 조회/삭제 화면 구현
* [SAC] SAC Report 항목 추가
* [SAC] External API를 통해서 Windows Server를 등록할 수 있도록 개선
* [KAC] Policy Editor 사용성 개선
* [WAC] Policy Editor 사용성 개선
