# 11.6.0

### QueryPie 11.6.0 Release

2026년 02월 13일


**New Feature**

* [공통] AI Chat 기본 기능 추가 (MCP Tool 호출, 대화 히스토리 관리 지원 포함)
* [DAC] AWS Secrets Manager 연동 지원 - DB 커넥션의 비밀번호를 AWS Secrets Manager에서 자동 조회하도록 Secret Store 기능 추가
* [DAC] Query Rule 기능 추가 - SQL 쿼리 패턴 기반 허용(ALLOW)/거부(DENY)/스킵(SKIP) 필터링 규칙 설정 지원
* [DAC] Custom JDBC Vendor 추가 - 사용자 정의 JDBC 드라이버 업로드 및 Custom JDBC Configurations 관리 기능 지원, ANSI SQL 레벨 쿼리 분석 지원


**Improvement**

* [공통] OAuth Client 관리 개선 - scope 선택, 다중 redirect URL, 상세 페이지 도입 및 DCR endpoint 타입 안정성 강화
* [공통] Email Integration 워크플로우 알림 비활성화 시에도 이메일이 발송되던 문제 수정
* [공통] macOS Multi-Agent 설치 시 /usr/local/bin 디렉토리 미존재로 qpctl 명령어를 찾을 수 없던 문제 수정
* [DAC] MySQL LOAD DATA LOCAL INFILE 대용량 파일 처리 개선
* [DAC] Trino 프록시 연결 시 불필요한 메타데이터 조회 쿼리 개선
* [DAC] Julian/Gregorian 달력 차이로 인한 SAP HANA SECONDDATE 오류 해결
* [DAC] SQL Request Workflow에서 Aborted 상태 확인 - 싱글톤 객체 오염으로 인한 중복 실행 버그 수정
* [DAC] PostgreSQL RETURNING 구문에 대한 정책 지원 추가
* [DAC] SAP HANA BLOB 타입 컬럼 조회 시 minWritableBytes 오류 수정
* [DAC] SAP BTP 웹에디터 주석 방식을 ANSI SQL로 변경
* [DAC] Web Editor 컨텍스트 메뉴 DDL 작업(Truncate/Drop/Rename) Audit Log 미기록 수정
* [DAC] Policy Audit Log에서 로그가 WHERE 조건 개수만큼 중복 기록되는 문제 수정
* [DAC] Reverse Tunnel Server Redis TLS 지원 및 클러스터 모드 자동 감지 개선
* [DAC] Reverse Tunnel을 통한 Redis Cluster 접속 불가 현상 수정
* [DAC] HTTPS 인증서 미설정 시 명확한 에러 메시지 출력하도록 개선
* [DAC] New DAC Policy Management 비활성 시 Workflow Tag Display 기능 미노출 이슈 수정
* [SAC] Server Agent 코드 서명 검증 로직 개선 - 인증서 변경 호환성 확보
* [SAC] Azure Cloud Provider 동기화 시 리소스 경로 대소문자 불일치로 VM 누락되던 문제 수정
* [SAC] Cloud Provider 동기화 시 IndexOutOfBoundsException 발생 문제 수정
* [WAC] Root CA Certificate 업로드 시 Key Pair 검증 로직 추가
