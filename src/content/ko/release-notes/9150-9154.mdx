## QueryPie 9.15.0 Release

2023년 12월 08일


**New Features**

* [Security] 패스워드 만료주기 설정 및 초기화 기능 추가
* [Workflow] 백그라운드 쿼리 실행
* [Audit] Log Export 메뉴 신규 추가
* [DAC] Secret Store 연동 기능 추가 및 DB 인증정보에 Vault 연동 지원
* [DAC] Snowflake 커넥션 연결 지원


**Enhancement**

* [Web Editor] Preview 처리 UX 개선
* [IDP] LDAP 동기화시 2개 이상 그룹 연동 지원
* [Agent] 에이전트 내에 QueryPie Host 선택 목록 추가
* [Agent] 에이전트 로그인 타임아웃 갱신 로직 개선
* [DAC] Redis SSL/TLS 지원
* [SAC] SSH 인증서 추가 대응


**Bug Fix**

* [Athena] AwsDataCatalog로 데이터베이스를 변경 할 수 없음
* [MongoDB] Import 시 '"undefined" is not valid JSON' 이라는 에러가 발생
* [Workflow] SQL Request 파일 타입 요청건 재상신시 에디터모드 사용할 수 없는 이슈
* [Editor] Querypie Export, mysqldump 이슈 해결
* [Agent] 커넥션 목록의 Expose 옵션 처리 반영되도록 수정
* [Athena] IAM 권한으로 인한 오류 발생시 DB 연결 끊어지는 이슈 개선
