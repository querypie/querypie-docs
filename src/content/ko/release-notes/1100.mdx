## QueryPie 11.0.0 Release

2025년 07월 02일


**New Feature**

* [General] 관리자 페이지 IP ACL 지원
* [General] 버전 표기 변경
* [General] 로그인 시 IP 접근 제한 강화 및 사용자 IP 신청 Workflow 추가
* [General] 대리 상신 기능 추가 (DB Access / Unmasking / Restricted Data Access / DB Policy Exception)
* [General] Attribute 기반 Workflow 메뉴 비노출
* [DAC] DB Access Request에서 태그로 요청 대상 필터링
* [DAC] 특정 사용자 기반 DB Access Request 필터링 설정
* [DAC] Redshift 커넥션에서 사용자 식별을 위한 사용자 정의 주석 추가 기능
* [DAC] Data Access / Data Masking / Sensitive Data 이벤트에 대한 syslog 전송 지원
* [DAC] Workflow에 Data Access 제한 해제 요청 타입 추가
* [DAC] Workflow에 DB Policy Exception Request 추가
* [DAC] Workflow를 통한 정책 예외 관리 및 관리자의 수동 예외 처리 기능
* [DAC] Sensitive Data 접근 모니터링 정책 구현
* [DAC] Policy Type에 DML Query Approval 추가
* [DAC] Policy Audit Log 기록
* [DAC] Audit Log Export 항목 추가 (Workflow Unmasking / Restricted Data Access / DB Policy Exception Request)
* [SAC] JIT 임시계정 생성/접속/종료 후 회수 지원
* [SAC] RDP 접속 종료 시, 로그인 유지 옵션 추가
* [SAC] 사용자의 서버 세션 종료 전 Agent 알림 기능
* [WAC] Audit Log Export 항목 추가 (Web App Access History / Web App Event Audit / Web App Role History / JIT Access Control Logs)

**Improvement**

* [General] 사용자 대시 보드에서 ADMIN 문구 제거
* [General] User Group &gt; Detail 드로우에서 Email 검색 지원
* [General] Okta 연동시 QueryPie의 Display name이 옥타 display name으로 변경
* [General] Okta, Onelogin, SAML IDP를 연동한 상태에서, 로컬 계정 MFA 지원
* [General] Audit Log Export 시 Workflow에 Target User 추가
* [General] Auth Provider 상관없이 최소 1명의 Owner 계정이 반드시 존재하도록 개선
* [General] 멀티에이전트 네트워크 끊어졌다가 다시 연결되었을 때 자동으로 Retry 하도록 개선
* [General] IP block 페이지에서 로그인 페이지로 이동할 수 있는 링크 제공
* [DAC] Default privilege 자동 설정
* [DAC] 정책이 설정된 커넥션은 Query Audit을 On으로 강제화
* [DAC] Proxy를 통한 Athena 쿼리 결과 조회 개선
* [DAC] Data Scope 에서 경로로 지정
* [DAC] 태그 검색 시 부분 일치 검색이 가능하도록 검색 방법 개선
* [SAC] Web / Agent에서 분단위로 받은 권한(Access, Privilege)를 받은 항목 표시
* [WAC] User Page &gt; Web App &gt; JIT Role 선택 후, 특정 Web App의 "..." 클릭시 레코딩 되는 항목 표시
