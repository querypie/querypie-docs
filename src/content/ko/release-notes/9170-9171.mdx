## QueryPie 9.17.0 Release

2024년 03월 11일


**New Features**

* [Audit] External API V2 및 Document 신규 출시
* [Audit] 감사로그 내보내기 Activity Logs, DML Snapshot 및 SAC 감사로그 범위로 지원 영역 확대 
* [DAC] Custom Identity Provider을 통한 인사정보 파일 연동 지원 
* [DAC] DocumentDB Proxy 지원
* [DAC] Tibero 신규 지원
* [DAC] Privilege 권한별 DB 계정 통제 기능 Advanced Privilege Setting 추가 
* [DAC] 워크플로 신청 시 권한 만료 기한에 최댓값 설정 기능
* [DAC] 커넥션 레벨 사유 입력 기능 제어 기능 추가
* [SAC] Proxy를 통한 3rd-party 클라이언트 접속 지원
* [SAC] Server Tag에 따른 Server Group 그룹핑 지원 


**Enhancement**

* [DAC] MongoDB + SRV 지원 
* [SAC] Policy 오브젝트화 구조 개선 
* [SAC] Web Terminal / Command Audit 로깅 강화 
* [Security] 2FA 초기화 로직 개선
* [Security] API Token 페이지 보안 강화 개선 


**Bug Fix**

* [Audit] Activity Logs 일부 Update 로그 Affected Data 대상 확인 불가 이슈 해결
* [Oracle] MERGE INTO 를 통한 DML 쿼리 실행시 DML Snapshot 남지 않는 이슈 해결 
* [Security] 모의해킹을 통한 내부 발견 보안 취약점 일괄 조치 완료
* [Audit] Kotlin 호환 이슈로 Workflow External API V2 내 boolean 속성 키명 isXXX에서 변경 처리 완료
    * Workflow - DB Access Request Details
        * accessControls[].connection.deleted
        * accessControls[].privilege.deleted
        * approvalSteps[].assignees[].delegated
    * Workflow - SQL Request Details
        * connection.deleted
        * approvalSteps[].assignees[].delegated
    * Workflow - SQL Export Request Details
        * connection.deleted
        * approvalSteps[].assignees[].delegated
