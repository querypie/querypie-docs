## QueryPie 10.3.0 Release

2025년 05월 14일


**New Feature**

* [General] 사용자 동시 로그인 제한 기능 추가
* [General] Custom Attribute 추가하는 기능 추가
* [General] LDAP User Attribute를 QueryPie에 연동하는 기능 추가 
* [General] Workflow에서 User Attribute 기반의 승인자 지정 기능 추가
* [General] Vault AppRole 인증 방식 추가
* [General] Privilege Request에서 Server Group Owner 승인 기능 추가
* [DAC] Cloud Provider 동기화 시 자동 태깅 기능 추가
* [DAC] DB 접속시 Reverse Tunnel 지원 (NOVA) 하도록 개선 
* [DAC] HBase 신규 지원 
* [DAC] 복호화 워크플로 및 CubeOne 연동 
* [DAC] 웹에디터에서 Data Export 시도시 DRM 연동 
* [DAC] 수행중인 Query를 관리자가 중지할 수 있는 기능 제공 
* [DAC] 웹에디터에서 Data Export 시도시 DRM 연동 
* [DAC] Multi-Agent에서 DAC의 Default Privilege 선택
* [SAC] Server Group에서 개인 계정 변수 지정 기능 구현
* [SAC] 사용자별 표시되는 Server Group 제한을 위한 Member 기능 추가
* [SAC] Server Grant Permission에서 시간단위 권한 부여 기능 추가
* [SAC] SSH 접속 시 VSCode 툴을 통해 접속할 수 있도록 지원
* [WAC] Web App Just-in-time Access Request 기능 추가
* [WAC] Web App 상세 페이지에서 웹앱 별 Owner 및 Member 설정 기능


**Enhancement**

* [General] 사용자 대시보드 권한 회수 시 Recent Connections에 표기되는 방법 개선
* [General] Audit Log Export Preview UI 개선건
* [General] Configuration 설정을 Approval Rules -&gt; General 메뉴로 이동
* [General] Workflow의 Server Access Request 신청시 시간단위 신청 기능 추가
* [General] Workflow의 Request 타입별 사용자 노출 제어 기능
* [General] Workflow에서 결재자의 자가 승인 허용 여부 옵션 추가
* [DAC] Security 설정을 General &gt; Security -&gt; Databases &gt; Configurations 메뉴로 이동
* [SAC] Role 변경 및 Direct Permission 부여되는 경우에도 세션 유지하도록 개선
* [SAC] Server Privilege Request의 신청 시간 기본값 60분으로 수정
* [SAC] Security 설정을 General &gt; Security -&gt; Servers &gt; Configurations 메뉴로 이동
* [KAC] Security 설정을 General &gt; Security -&gt; Kubernetes &gt; Configurations 메뉴로 이동
* [WAC] Web App 생성 시 Base URL로 IP주소 입력할 수 있도록 개선


**Bug Fix**

* [General] RDP Server Agent 버전 표기가 올바르게 되지 않던 이슈 해결
* [SAC] Terminal을 열고 Role이 변경된 경우, 터미널의 "+" 클릭시 Servers 탭으로 이동하도록 수정
* [WAC] Slack에서 승인 후 링크 클릭시 접속 불가 이슈 해결
* [WAC] 삭제된 유저가 부여한 role 회수 시 에러 해결
* [WAC] base url에 더미값을 넣은 경우 에러 발생 이슈 해결
* [WAC] 유저 대시보드 Web App 이름이 길어질 때 정렬 안되는 버그 수정
* [WAC] QueryPie 로그인 풀린 상태에서 익스텐션 Pop-up의 로그인 안내 미노출 이슈 해결
* [Workflow] Urgent mode Off 설정시 Web App Just-In-Time Access Request에서 Urgent mode 설정 영역 노출되는 이슈 해결
