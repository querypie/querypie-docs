## **QueryPie 9.16.0 Release**

2024. 1. 26.

**New Features**

* [Security] 사용자별 IP 통제 기능
* [Audit] 감사로그 통계 리포트 Beta 버전 출시
* [Audit] 로그 무결성 검증 기능 추가

**Enhancement**

* [DAC] Cloud Provier Azure 연결 지원
* [DAC] MongoDB 멀티 호스트 방식 지원
* [SAC] Tag 기반 서버 그룹 자동 추가 기능 지원
* [SAC] Secret Store 연동 기능 추가 및 서버 인증정보에 Vault 연동 지원
* [SAC] SSH Key 파일 업로드 방식 지원
* [Audit] DML Snapshot 로그 before/after 데이터에 민감정보 마스킹 처리 개선

**Bug Fix**

* [MySQL] LEAD 함수 안에서 IF 문 사용하는 경우 오류 해결
* [MariaDB] MariaDB 10.6.2.0 버전 이상인 경우 TablePlus 접속 이슈 해결
* [MongoDB] Aggregate 구문 실행 관련 오류 해결
* [MongoDB] 프록시 접속시 Data Compression 옵션 관련 개선
