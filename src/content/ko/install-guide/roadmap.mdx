# 🏃‍♀️‍➡️ Roadmap

QueryPie AI Hub는 사용자에게 더 나은 AI 통합 경험을 제공하기 위해 끊임없이 노력하고 있습니다.
LLM Host로서의 완성도를 높여, 더욱 정확한 답변을 제공할 수 있도록 지속적으로 개선할 예정입니다.
또한, 보다 안전한 MCP 및 Agent 환경을 위해 보안 기능들을 지속적으로 추가할 예정입니다.

## 협업 및 조직 관리 🏢
팀 단위의 효율적인 AI 활용을 위한 협업 환경을 구축합니다.

- 🔴 **조직(Organization)** `📅 2025년 3분기`: 모든 사용자가 조직을 생성하고, 조직 내에서 협업을 할 수 있도록 지원합니다. 조직 내 사용자 초대, 프롬프트 및 툴 프리셋 공유 등 다양한 협업 기능을 제공합니다.
- 🔴 **역할 기반 접근 제어** `📅 2025년 4분기`: 조직 내에서 관리자, 편집자, 뷰어 등 세분화된 권한 관리를 통해 보안성과 효율성을 동시에 확보합니다.


## MCP 및 통합 확장 🔗
업무 자동화와 생산성 향상을 위한 다양한 서비스 통합을 지원합니다.

- 🟡 **MCP Integration 확장** `📅 2025년 2분기 ~ 지속`: AI 업무 자동화에 필수적인 SaaS 앱을 지속적으로 확장하여 원활한 워크플로우를 구축할 수 있도록 지원할 예정입니다.

  - Salesforce
  - ServiceNow
  - Hubspot
  - Dropbox
  - Airtable
  - Jenkins
  - PagerDuty
  - OpsGenie
  - Box
  - Zoom
  - Docusign

  - LlamaIndex
  - Context7

- 🟡 **Bring Your Own MCP Integration** `📅 2025년 3분기`: SSE/Streamable HTTP 연동 및 인증 방식을 지원하여, 기업 내부에 이미 구축된 MCP 서버와 QueryPie AI Hub를 연결할 수 있도록 지원할 예정입니다.

## 보안 및 인프라 🔒
기업 환경에서 안전하게 AI를 활용할 수 있는 보안 기능을 강화합니다.

- 🟡 **Tunneling** `📅 2025년 3분기`: 사용자 네트워크에 접근하기 위하여 사용자 로컬 프록시를 통해 터널링 기능을 제공할 예정입니다. 자세한 내용은 [터널링](/user-guide/tunneling#roadmap-tunneling) 문서를 참고해주세요.
- 🔴 **Audit Log** (Beta) `📅 2025년 3분기`: 조직 내 모든 사용자 행위를 상세히 기록하고, 이를 통해 컴플라이언스 요구사항을 충족하고 보안 사고 추적을 지원합니다.
- 🔴 **DLP (Data Loss Prevention)** `📅 2025년 3분기`: 사내의 민감 데이터를 보호하기 위하여, 사용자의 입력 데이터 또는 LLM의 응답 데이터에서 민감 정보를 실시간으로 검출하고 추가적인 조치의 유형을 설정할 수 있도록 합니다.
- 🔴 **UEBA (User and Entity Behavior Analytics)** `📅 2025년 3분기`: 사용자의 행위 패턴을 AI로 분석하여, 이상행위나 보안 위협을 조기에 감지하고 자동으로 차단하거나 관리자에게 알림을 발송합니다.
- 🔴 **LLM Guardrails** `📅 2025년 3분기`: LLM에 대한 Prompt Injection, 민감 정보 노출, 부적절한 행위 등을 방지하기 위한 보안 기능을 제공합니다.

## AI 성능 및 사용자 경험 🚀

더욱 정확하고 효율적인 AI 서비스를 위한 지속적인 개선을 진행합니다.

- 🟡 **컨텍스트 인식 개선** `📅 2025년 2분기 ~ 지속`: 대화 히스토리와 사용자 패턴을 학습하여 더욱 정확하고 개인화된 응답을 제공합니다.
- 🟡 **성능 모니터링** `📅 2025년 2분기 ~ 지속`: 실시간 응답 속도, 정확도 지표, 그리고 사용자 만족도를 추적하여 서비스 품질을 지속적으로 개선합니다.
- 🔴 **Unified MCP Tool** `📅 2025년 3분기`: 연동된 MCP 툴의 개수가 늘어나더라도 저비용 고성능으로 MCP 툴 호출이 가능하도록 지원합니다.

## 미래 비전 🔭

QueryPie AI Hub는 단순한 AI 도구를 넘어서, 기업의 디지털 트랜스포메이션을 이끄는 핵심 플랫폼이 되고자 합니다.

우리는 AI가 일상적인 업무를 자동화하고, 창의적이고 전략적인 업무에 더 많은 시간을 할애할 수 있는 미래를 그리고 있습니다. 데이터 분석부터 문서 작성, 고객 응대까지 - AI가 여러분의 든든한 업무 파트너가 되어 더 높은 가치를 창출할 수 있도록 지원하겠습니다.
