# Kakao Map

카카오 지도 API를 통해 장소 검색, 주소 변환, 좌표 변환 등의 지도 관련 기능을 사용할 수 있습니다.

## 주요 기능
- 키워드 기반 장소 검색
- 카테고리별 장소 검색
- 주소를 좌표로 변환
- 좌표를 주소로 변환
- 좌표를 행정구역 정보로 변환
- 좌표계 간 변환

## 기본 정보

### 인증 방식
API Key를 사용하여 인증합니다.

### 입력 파라미터
- **API Key** (필수): 카카오 개발자 센터에서 발급받은 REST API 키

## 지원하는 MCP 도구

### 장소 검색 도구
#### search_places
키워드를 기반으로 카카오 지도에서 장소를 검색합니다.

**입력 파라미터:**
- `query` (필수): 검색할 키워드
- `x` (선택): 중심 좌표의 X값 (경도)
- `y` (선택): 중심 좌표의 Y값 (위도)
- `radius` (선택): 중심 좌표부터의 반경거리 (미터)
- `page` (선택): 결과 페이지 번호 (1~45)
- `size` (선택): 한 페이지에 보여질 문서의 개수 (1~15)

#### map_category_search
카테고리별로 카카오 지도에서 장소를 검색합니다.

**입력 파라미터:**
- `category_group_code` (필수): 카테고리 그룹 코드
- `x` (선택): 중심 좌표의 X값 (경도)
- `y` (선택): 중심 좌표의 Y값 (위도)
- `radius` (선택): 중심 좌표부터의 반경거리 (미터)
- `page` (선택): 결과 페이지 번호
- `size` (선택): 한 페이지에 보여질 문서의 개수

### 좌표 변환 도구
#### map_address_to_coord
주소를 좌표로 변환합니다.

**입력 파라미터:**
- `query` (필수): 주소 검색어

#### map_coord_to_address
좌표를 주소로 변환합니다.

**입력 파라미터:**
- `x` (필수): X 좌표 (경도)
- `y` (필수): Y 좌표 (위도)

#### map_coord_to_region_code
좌표를 한국 행정구역 정보로 변환합니다.

**입력 파라미터:**
- `x` (필수): X 좌표 (경도)
- `y` (필수): Y 좌표 (위도)

#### map_trans_coord
좌표계 간 좌표를 변환합니다.

**입력 파라미터:**
- `x` (필수): X 좌표
- `y` (필수): Y 좌표
- `input_coord` (필수): 입력 좌표계
- `output_coord` (필수): 출력 좌표계

## 사용 예시

예를 들어,
- Prompt로 "강남역 근처 카페를 찾아줘" 라고 입력하면,
- LLM은 MCP를 통해 `search_places` 도구를 호출하여 강남역 근처 카페를 검색합니다.
- 그리고, Tool 호출 결과를 기반으로 채팅에서 응답합니다.

## API Key 설정 방법
1. 카카오 개발자 센터에서 애플리케이션을 등록합니다.
2. REST API 키를 발급받습니다.
3. QueryPie AI에서 Kakao Map 통합을 선택합니다.
4. 발급받은 API 키를 입력하여 연결을 완료합니다.

## 주의사항
- 생성한 API Key는 안전하게 보관하세요.
- 일일 호출 한도를 확인하고 적절히 사용하세요.
- 좌표는 WGS84 좌표계를 기본으로 사용합니다.
