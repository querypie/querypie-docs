# 🧩 MCP Server Integration

MCP(Model Context Protocol)는 AI 모델이 외부 도구에 연결하는 방식을 표준화하는 프로토콜입니다.

QueryPie AI Hub에서는 자체적으로 MCP 서버를 제공하며, 이를 통해 다양한 도구와 데이터 소스에 안전하게 연결할 수 있습니다.

## Workflow Automation
워크플로우 자동화 도구를 연동하고, 순차적 사고 도구를 활용하여 복잡한 문제를 해결할 수 있습니다.

- [Sequential Thinking](./mcps/sequentialthinking) - 순차적 사고 도구
- [n8n Chat Trigger Node](./mcps/n8n-chat-trigger-node) - n8n 워크플로우 트리거
- [Dify API Access](./mcps/dify-api-access) - Dify AI 플랫폼 연동

## Communication & Collaboration
커뮤니케이션 및 협업 도구와 연동하여, 협업을 가속화할 수 있습니다.

- [Google Gmail](./mcps/google-gmail) - 이메일 관리
- [Slack](./mcps/slack) - 팀 커뮤니케이션
- [Discord](./mcps/discord) - 음성 및 텍스트 채팅
- [GitHub](./mcps/github) - Git 저장소 및 이슈 관리

## Google Services
구글 드라이브, 스프레드시트, 캘린더 등 구글 서비스와 연동하여, 다양한 업무를 자동화할 수 있습니다.

- [Google Drive](./mcps/google-drive) - 클라우드 파일 저장소
- [Google Sheets](./mcps/google-sheets) - 스프레드시트 관리
- [Google Calendar](./mcps/google-calendar) - 일정 관리

## Project Management
Atlassian 팀 협업 도구와 연동하여, 프로젝트 관리를 효율화할 수 있습니다.

- [Atlassian Confluence](./mcps/atlassian-confluence) - 팀 협업 및 문서화
- [Atlassian Jira](./mcps/atlassian-jira) - 이슈 추적 및 프로젝트 관리

## Development & DevOps
개발 및 운영에 필요한 도구와 연동하여, 개발 및 운영을 효율화할 수 있습니다.

- [AWS](./mcps/aws) - Amazon Web Services 통합
- [Code Executor](./mcps/code-executor) - 코드 실행 환경
- [Grafana](./mcps/grafana) - 모니터링 및 시각화
- [Kubernetes](./mcps/kubernetes) - 컨테이너 오케스트레이션
- [SSH](./mcps/ssh) - 원격 서버 접속
- [QueryPie Client](./mcps/querypie-client) - QueryPie 클라이언트
