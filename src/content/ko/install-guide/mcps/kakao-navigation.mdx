# Kakao Navigation

카카오 내비게이션 API를 통해 경로 안내, 길찾기, 경유지 설정 등의 내비게이션 기능을 사용할 수 있습니다.

## 주요 기능
- 출발지에서 목적지까지의 경로 안내
- 경유지를 포함한 경로 계획
- 다중 출발지 경로 계산
- 다중 목적지 경로 계산
- 미래 시간 기준 경로 계산

## 기본 정보

### 인증 방식
API Key를 사용하여 인증합니다.

### 입력 파라미터
- **API Key** (필수): 카카오 개발자 센터에서 발급받은 REST API 키

## 지원하는 MCP 도구

### 경로 안내 도구
#### navigation_directions
출발지에서 목적지까지의 경로를 계산합니다.

**입력 파라미터:**
- `origin` (필수): 출발지 좌표 (경도,위도)
- `destination` (필수): 목적지 좌표 (경도,위도)
- `priority` (선택): 경로 우선순위 (RECOMMEND, TIME, DISTANCE)
- `car_fuel` (선택): 차량 연료 타입
- `car_hipass` (선택): 하이패스 사용 여부
- `alternatives` (선택): 대안 경로 제공 여부
- `road_details` (선택): 도로 상세 정보 제공 여부

#### navigation_waypoints
경유지를 포함한 경로를 계산합니다.

**입력 파라미터:**
- `origin` (필수): 출발지 좌표
- `destination` (필수): 목적지 좌표
- `waypoints` (필수): 경유지 좌표 목록
- `priority` (선택): 경로 우선순위
- `car_fuel` (선택): 차량 연료 타입
- `car_hipass` (선택): 하이패스 사용 여부

#### navigation_origins
다중 출발지에서 하나의 목적지까지의 경로를 계산합니다.

**입력 파라미터:**
- `origins` (필수): 출발지 좌표 목록
- `destination` (필수): 목적지 좌표
- `priority` (선택): 경로 우선순위
- `car_fuel` (선택): 차량 연료 타입
- `car_hipass` (선택): 하이패스 사용 여부

#### navigation_destinations
하나의 출발지에서 다중 목적지까지의 경로를 계산합니다.

**입력 파라미터:**
- `origin` (필수): 출발지 좌표
- `destinations` (필수): 목적지 좌표 목록
- `priority` (선택): 경로 우선순위
- `car_fuel` (선택): 차량 연료 타입
- `car_hipass` (선택): 하이패스 사용 여부

#### navigation_future
미래 시간을 기준으로 경로를 계산합니다.

**입력 파라미터:**
- `origin` (필수): 출발지 좌표
- `destination` (필수): 목적지 좌표
- `departure_time` (필수): 출발 시간
- `priority` (선택): 경로 우선순위
- `car_fuel` (선택): 차량 연료 타입
- `car_hipass` (선택): 하이패스 사용 여부

## 사용 예시
카카오 내비게이션 MCP 서버는 카카오 지도 MCP 서버와 함께 활용해야 합니다. <br/>
출발지에서 목적지까지 가는 경로를 찾기 위해서, 먼저 장소 쿼리를 좌표로 변환하고 이를 이용하여 경로를 계산합니다.

예를 들어,
- Prompt로 "강남역에서 홍대입구역까지 가는 길을 알려줘" 라고 입력하면,
- LLM은 카카오 지도 MCP 서버의 `map_address_to_coord` 도구를 먼저 호출하여 강남역의 좌표, 홍대입구역의 좌표를 얻습니다.
- 이어서 LLM은 카카오 내비게이션 MCP 서버의 `navigation_directions` 도구를 호출하여 경로를 계산합니다.
- 그리고, Tool 호출 결과를 기반으로 채팅에서 응답합니다.

## API Key 설정 방법
1. 카카오 개발자 센터에서 애플리케이션을 등록합니다.
2. REST API 키를 발급받습니다.
3. QueryPie AI에서 Kakao Navigation 통합을 선택합니다.
4. 발급받은 API 키를 입력하여 연결을 완료합니다.

## 주의사항
- 생성한 API Key는 안전하게 보관하세요.
- 일일 호출 한도를 확인하고 적절히 사용하세요.
- 좌표는 WGS84 좌표계를 사용합니다.
- 실시간 교통 정보를 반영한 경로를 제공합니다.
