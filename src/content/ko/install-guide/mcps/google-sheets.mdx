# Google Sheets

Google Sheets와 연결하여 스프레드시트 관리 및 데이터 처리를 수행할 수 있습니다.

## 주요 기능
- 스프레드시트 생성 및 관리
- 시트 데이터 조회 및 업데이트
- 셀 및 범위 데이터 배치 처리
- 시트 복사, 이름 변경, 생성
- 행/열 추가 및 관리
- 스프레드시트 공유 및 권한 설정
- 여러 스프레드시트 동시 처리

## 기본 정보

### 인증 방식
OAuth 2.0을 사용하여 인증합니다.

### 필요한 권한
- `https://www.googleapis.com/auth/spreadsheets`: Google Sheets 전체 접근 권한

## 지원하는 MCP 도구

### 데이터 조회
#### get_sheet_data
특정 스프레드시트의 시트에서 데이터를 조회합니다.

#### get_multiple_sheet_data
Google Sheets의 여러 특정 범위에서 데이터를 조회합니다.

#### get_multiple_spreadsheet_summary
여러 Google Sheets의 요약 정보를 조회합니다.

### 데이터 업데이트
#### update_cells
Google Sheets의 셀을 업데이트합니다.

#### batch_update_cells
Google Sheets의 여러 범위를 배치로 업데이트합니다.

### 시트 구조 관리
#### add_rows
Google Sheets의 시트에 행을 추가합니다.

#### add_columns
Google Sheets의 시트에 열을 추가합니다.

#### list_sheets
Google Sheets의 모든 시트 목록을 조회합니다.

#### copy_sheet
한 스프레드시트에서 다른 스프레드시트로 시트를 복사합니다.

#### rename_sheet
Google Sheets의 시트 이름을 변경합니다.

#### create_sheet
기존 Google Sheets에 새 시트 탭을 생성합니다.

### 스프레드시트 관리
#### create_spreadsheet
새 Google Sheets를 생성합니다.

#### list_spreadsheets
구성된 Google Drive 폴더의 모든 스프레드시트 목록을 조회합니다.

#### share_spreadsheet
이메일을 통해 여러 사용자와 Google Sheets를 공유합니다.

## 사용 예시

예를 들어,
- Prompt로 "스프레드시트의 A1:C10 범위 데이터를 조회해줘" 라고 입력하면,
- LLM은 MCP를 통해 `get_sheet_data` 도구를 호출하여 해당 범위의 데이터를 조회합니다.
- 그리고, Tool 호출 결과를 기반으로 채팅에서 응답합니다.
