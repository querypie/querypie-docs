# Code Executor

샌드박스 환경에서 LLM이 생성한 코드를 안전하게 실행할 수 있습니다.

## 주요 기능
- 샌드박스 환경에서 안전한 코드 실행
- 실행 결과 및 오류 메시지 제공
- 실시간 코드 테스트

### 사용 가능한 기능
- 기본 JavaScript 문법과 내장 객체들 (Array, Object, Math, Date 등)
- 함수 정의 및 호출
- 변수 선언 및 조작
- 조건문, 반복문 등 제어 구조
- 기본적인 계산 및 데이터 처리

### 활용 예시
- 수학 계산이나 알고리즘 테스트
- 데이터 변환 및 처리 로직 검증
- 간단한 함수나 클래스 구현 테스트

## 기본 정보

### 인증 방식
별도의 인증이 필요하지 않습니다.

### 입력 파라미터
추가로 입력해야 하는 파라미터가 없습니다.

## 지원하는 MCP 도구

### run_code
샌드박스 환경에서 JavaScript(ECMAScript) 코드를 실행하고 결과를 반환합니다.

## 주요 제약사항
- 외부 라이브러리 import 불가 - npm 패키지나 외부 모듈을 가져올 수 없습니다.
- 파일 시스템 접근 불가 - 파일 읽기/쓰기 작업은 불가능합니다.
- 네트워크 요청 제한 - fetch, XMLHttpRequest 등을 통한 외부 API 호출은 불가능합니다.
- DOM 조작 불가 - document, window 객체 등 브라우저 API 사용은 불가능합니다.
- 비동기 작업 제한 - setTimeout, setInterval 등의 타이머 함수 사용은 불가능합니다.
