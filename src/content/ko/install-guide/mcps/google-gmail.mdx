# Gmail

Google Gmail과 연결하여 이메일 전송, 조회, 관리 등을 수행할 수 있습니다.

## 주요 기능
- 이메일 전송 및 임시 저장
- 이메일 조회 및 검색
- 이메일 라벨 관리
- 이메일 삭제 및 수정
- 배치 작업 지원

## 기본 정보

### 인증 방식
OAuth 2.0을 사용하여 인증합니다.

### 필요한 권한
- `https://mail.google.com/`: Gmail 전체 접근 권한

## 지원하는 MCP 도구

### 이메일 전송 도구
#### send_email
새로운 이메일을 전송합니다.

#### draft_email
이메일을 임시 저장합니다.

### 이메일 조회 도구
#### read_email
특정 이메일의 내용을 조회합니다.

#### search_emails
Gmail 검색 구문을 사용하여 이메일을 검색합니다.

### 이메일 관리 도구
#### modify_email
이메일 라벨을 수정합니다. (폴더 이동)

#### delete_email
이메일을 영구 삭제합니다.

### 라벨 관리 도구
#### list_email_labels
사용 가능한 모든 Gmail 라벨을 조회합니다.

#### create_label
새로운 Gmail 라벨을 생성합니다.

#### update_label
기존 Gmail 라벨을 수정합니다.

#### delete_label
Gmail 라벨을 삭제합니다.

#### get_or_create_label
라벨을 조회하거나 없으면 생성합니다.

### 배치 작업 도구
#### batch_modify_emails
여러 이메일의 라벨을 배치로 수정합니다.

#### batch_delete_emails
여러 이메일을 배치로 영구 삭제합니다.

## 사용 예시

예를 들어,
- Prompt로 "받은 편지함의 최근 이메일을 조회해줘" 라고 입력하면,
- LLM은 MCP를 통해 `search_emails` 도구를 호출하여 이메일을 검색합니다.
- 그리고, Tool 호출 결과를 기반으로 채팅에서 응답합니다.
