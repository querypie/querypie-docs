# Daum Search

다음(Daum) 검색 API를 통해 웹 문서, 이미지, 블로그, 카페 등의 검색 기능을 사용할 수 있습니다.

## 주요 기능
- 웹 문서 검색
- 이미지 검색
- 블로그 포스트 검색
- 카페 게시글 검색

## 기본 정보

### 인증 방식
API Key를 사용하여 인증합니다.

### 입력 파라미터
- **API Key** (필수): 카카오 개발자 센터에서 발급받은 REST API 키

## 지원하는 MCP 도구

### 검색 도구
#### search_web
다음에서 웹 문서를 검색합니다.

**입력 파라미터:**
- `query` (필수): 검색할 질의어
- `sort` (선택): 결과 문서 정렬 방식 (accuracy, recency)
- `page` (선택): 결과 페이지 번호 (1~50)
- `size` (선택): 한 페이지에 보여질 문서의 개수 (1~50)

#### search_image
다음에서 이미지를 검색합니다.

**입력 파라미터:**
- `query` (필수): 검색할 질의어
- `sort` (선택): 결과 문서 정렬 방식 (accuracy, recency)
- `page` (선택): 결과 페이지 번호 (1~50)
- `size` (선택): 한 페이지에 보여질 문서의 개수 (1~80)

#### search_blog
다음에서 블로그 포스트를 검색합니다.

**입력 파라미터:**
- `query` (필수): 검색할 질의어
- `sort` (선택): 결과 문서 정렬 방식 (accuracy, recency)
- `page` (선택): 결과 페이지 번호 (1~50)
- `size` (선택): 한 페이지에 보여질 문서의 개수 (1~50)

#### search_cafe
다음에서 카페 게시글을 검색합니다.

**입력 파라미터:**
- `query` (필수): 검색할 질의어
- `sort` (선택): 결과 문서 정렬 방식 (accuracy, recency)
- `page` (선택): 결과 페이지 번호 (1~50)
- `size` (선택): 한 페이지에 보여질 문서의 개수 (1~50)

## 사용 예시

예를 들어,
- Prompt로 "최신 AI 기술에 대한 블로그 글을 찾아줘" 라고 입력하면,
- LLM은 MCP를 통해 `search_blog` 도구를 호출하여 AI 기술 관련 블로그 글을 검색합니다.
- 그리고, Tool 호출 결과를 기반으로 채팅에서 응답합니다.

## API Key 설정 방법
1. 카카오 개발자 센터에서 애플리케이션을 등록합니다.
2. REST API 키를 발급받습니다.
3. QueryPie AI에서 Kakao Search 통합을 선택합니다.
4. 발급받은 API 키를 입력하여 연결을 완료합니다.
