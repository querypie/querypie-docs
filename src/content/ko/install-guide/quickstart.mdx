# 🚀 Quickstart

QueryPie AI Hub를 시작하기 위한 빠른 가이드입니다. <br />
이 가이드를 따라하시면, 단 몇 분 안에 다양한 SaaS 앱을 연동하고, AI와의 채팅에서 활용할 수 있을 거에요.

## AI Hub 계정 만들기
가장 먼저, AI Hub 계정을 만들어 보겠습니다.

1. [QueryPie AI Hub 홈페이지](https://app.querypie.com)에 접속한 뒤, **Login** 버튼을 클릭합니다. 
2. 현재 Google, GitHub, 이메일 회원 가입을 지원합니다.
  - 구글 또는 GitHub 계정으로 가입하려면 Continue with Google / Continue with GitHub 버튼을 클릭합니다. 
  -  이메일로 회원 가입하려면, 이메일 및 비밀번호 입력창 하단 Sign up 버튼을 클릭하고, 이름, 이메일, 비밀번호를 입력합니다.
3. 추가 회원 정보를 입력하고 가입을 완료합니다.

<Callout type="info">
현재는 초대받은 이메일로만 회원 가입이 가능합니다. <br />
초대가 필요하다면 aihub@querypie.com 으로 문의해주세요.
</Callout>

## Integration 생성하기

Integration은 외부 서비스나 데이터베이스와 연결하는 기능이에요. <br />
예를 들어, Slack, JIRA, Notion 등 다양한 서비스를 MCP를 통해 사용할 수 있게 해줍니다.

<Steps>
### Integration 선택하기
**Integrations** 메뉴의 **All Integrations** 탭을 클릭한 뒤, 연동하려는 앱 카드를 클릭합니다.

![Integration Page](@/app/img/integration-page.png)

### Integration 정보 입력하기
**Install New Integration** 페이지에서 연동 추가 정보를 입력합니다. 
   - **Name**: 연동 이름
   - **Description**: 연동 설명
   - **Integration Type**에 따라, 그 외 추가 정보 입력이 필요할 수 있습니다.
   - 이름과 설명을 용도에 맞게 입력하면 Tool 호출 정확도가 향상됩니다.
   
### OAuth 지원 앱 설치
Google, Atlassian 등 OAuth 인증이 지원되는 MCP 서버의 경우, Connect & Install 버튼을 클릭하면 해당 애플리케이션의 로그인/인증 페이지로 이동합니다.

  ![Integration-OAuth](@/app/img/integration-oauth-01.png)

로그인 페이지가 열리면, AI Hub에서 사용하려는 계정을 선택합니다. 

  ![Integration-OAuth](@/app/img/integration-oauth-02.png)

인증 이후 MCP 서버 설치가 완료됩니다.

### API Key 지원 앱 설치
AWS MCP와 같이 OAuth 인증을 지원하지 않는 MCP 서버의 경우, Access Key, Secret Key와 같은 API Key를 획득하여 입력해야 합니다. 모든 API Key 정보는 암호화되어 저장됩니다. <br />
**Install** 버튼을 클릭하면 입력한 API Key로 연동이 완료됩니다.

   ![Integration-API-Key](@/app/img/integration-aws.png)


### n8n Workflow 설치
기존에 만들어 놓은 워크플로우에서 Chat Trigger Node를 선택한 뒤, Chat URL과 인증정보를 복사합니다.

   ![Integration-n8n-01](@/app/img/integration-n8n-04.png)

Integration 페이지에서 n8n Chat Trigger Node 타입을 선택하고, 앞에서 복사한 정보를 입력한 뒤 저장합니다.

   ![Integration-n8n-03](@/app/img/integration-n8n-03.png)

</Steps>

## MCP Preset 생성하기

**MCP Preset**은 AI가 특정 컨텍스트에서 작동하도록 설정하는 템플릿입니다.

**MCP Presets** 탭으로 이동한 다음, **Create New Preset** 버튼을 클릭합니다. (프리셋을 최초로 생성한다면, 자동으로 프리셋 생성 페이지로 진입합니다.)

![MCP Preset](@/app/img/preset-create.png)

1. 프리셋 기본 정보를 입력합니다.
   - **MCP Preset Name**: 프리셋을 구분할 수 있는 이름입니다.
   - **Instruction**: 프리셋을 사용하여 채팅할 때, 기본적으로 포함되는 지시사항입니다. 프리셋의 역할, 컨텍스트, 응답 스타일 등을 포함할 수 있습니다.
2. **MCP Servers & Tools**에서 이 프리셋에 포함할 MCP 서버의 툴을 선택합니다.
3. **Save Changes** 버튼을 클릭하여 저장합니다.

<Callout type="info">
다양한 MCP 툴을 하나의 프리셋으로 묶어서 사용하면, 여러 SaaS에서 워크플로우를 수행할 수 있어요.
</Callout>

## AI Hub에서 채팅하기

이제 모든 설정이 완료되었습니다. MCP Preset을 사용하여 AI와 채팅을 시작해보세요!

1. **Chat** 탭으로 이동합니다.
2. `@` 를 입력한 다음, 컨텍스트 메뉴에서 방금 생성한 MCP Preset을 선택합니다.
3. 채팅창에 질문이나 요청을 입력합니다. 예를 들어 Slack MCP를 연동했다면, 다음과 같은 요청이 가능합니다.
   ```
   "helloworld 채널에 '테스트'라는 메시지를 보내주세요."
   ```
4. AI가 설정된 Integration을 통해 실시간으로 요청을 처리합니다.

더 많은 MCP 서버 기능이 궁금하다면 MCPs 문서를 확인해보세요.

<Callout type="info" emoji="🎉">
축하합니다! 이제 QueryPie AI Hub의 기본적인 사용법을 알게 되었습니다. <br /> 
MCP 서버 별로 제공하는 상세한 기능이 궁금하다면 [Available MCP Servers](/user-guide/mcps) 문서를 확인해보세요.
</Callout>
