# 🛠️ Supported LLM Apps

QueryPie AI Hub는 다양한 LLM 애플리케이션을 지원합니다. <br/>
MCP 프리셋별로 제공되는 로컬 프록시 설치 명령어를 실행하면, 원하는 LLM 애플리케이션에 설치할 수 있습니다.

## 지원하는 LLM 애플리케이션

- Cursor
- Claude Desktop
- Windsurf

## 사용 방법

![3rd Party Installation](@/app/img/3rd-party.png)

1. MCP 프리셋을 선택합니다.
2. Installation 탭에서 Generate Command 버튼을 클릭합니다.
    - 생성된 로컬 프록시 설치 명령어는 10분간 유효하며, 1회만 사용할 수 있습니다.
3. 터미널을 열고 로컬 프록시 설치 명령어를 실행합니다.
4. 로컬 프록시 설치가 완료되면, LLM 애플리케이션에서 QueryPie AI Hub의 MCP 프리셋을 사용할 수 있습니다.

## 주의사항

- 설치 명령어는 10분간 유효하며, 1회만 사용할 수 있습니다.
- 실제 인증에 사용되는 CLI Token은 MCP Configuration에 저장됩니다. CLI Token이 유출되지 않도록 주의하세요.
  - 발급된 CLI Token은 프로필 > 설정 > CLI Token 메뉴에서 삭제할 수 있습니다.
- MCP Preset의 Instruction은 3rd Party LLM 애플리케이션에서 동작하지 않습니다.
- 최대로 연동 가능한 툴의 개수는 LLM 애플리케이션의 제약사항에 따릅니다.
    - Cursor: 40개
    - Claude Desktop: 10개
    - Windsurf: 100개
