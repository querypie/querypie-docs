# 🌟 What is QueryPie AI Hub?

QueryPie AI Hub는 MCP(Model Context Protocol) 서버를 조합하여 다양한 에이전트를 만들고, 즉각적으로 구동할 수 있는 AI Playground 입니다.

n8n, Dify 등 Workflow tool 뿐만아니라, Slack, JIRA, Discord등 다양한 SaaS 애플리케이션을 MCP를 통해 사용할 수 있습니다.


## QueryPie AI Hub의 주요 기능

### 🚀 손쉬운 MCP 서버 연동
업무에 바로 적용해볼 수 있도록 엄선된 MCP 서버를 제공합니다. <br />

- AWS
- n8n
- Dify
- Slack
- Google Calendar, Gmail, Google Sheets 등등
- Jira, Confluence

이외에도 다양한 MCP 서버를 제공합니다. [Available MCP Servers](/user-guide/mcps) 문서를 확인해보세요.

### 🔧 MCP 프리셋 관리
자주 활용하는 MCP 서버와 Instruction을 프리셋으로 저장할 수 있습니다.<br/>

### 💬 LLM 채팅
프리셋을 멘션하여 채팅을 시작할 수 있습니다. <br/>
현재 제공 중인 모델은 Claude 4 Sonnet입니다.


[🚀 Quickstart](/user-guide/quickstart) 문서를 따라하면, 단 몇 분 만에 자주 쓰는 업무용 애플리케이션을 연동할 수 있습니다.


## FAQ
자주 묻는 질문들을 미리 정리했습니다. 혹시 궁금하신 점이 있으시다면 언제든 aihub@querypie.com 으로 질문해 주세요.

### 저희 회사는 QueryPie를 쓰고 있어요. AI Hub와 어떻게 연결되는 건가요?
QueryPie를 사용하고 계시군요! 감사합니다. 🙇‍♀️<br/>
QueryPie AI Hub는 DAC/SAC 유저 콘솔에 해당하는 QueryPie Client MCP 서버를 제공하고 있습니다.<br/>
QueryPie를 통해 직접 사내 리소스에 접속하는 대신, AI Hub을 통해 LLM에게 자연어로 요청하고, 결과를 받아볼 수 있어요.<br/>
자세한 가이드는 [QueryPie Client MCP 서버](/user-guide/mcps/querypie-client) 문서를 확인해보세요.

### 현재 제공하는 모델은 무엇인가요?
현재 제공하는 모델은 Claude 4 Sonnet입니다.<br/>
다른 모델을 제공하기 위해서 내부적으로 열심히 테스트 하고 있습니다 🏃‍♀️‍➡️

### 제가 가지고 있는 API를 사용할 수 없나요?
현재는 사용자 API 연동 기능을 제공하지 않고 있습니다. 

### 얼만큼 사용할 수 있어요?
CBT 기간 동안에는 사용량 제한 없이 LLM을 호출해 볼 수 있습니다.<br/>
많은 테스트 및 의견 부탁드려요. 🙏

### 앞으로 추가될 MCP 서버는 어떤게 있어요?

- Salesforce
- Notion
- ServiceNow
- LlamaIndex

이외에도, 다양한 MCP 서버가 계속 추가될 예정입니다. 어떤 MCP 서버가 추가될 예정인지 궁금하시다면 [Roadmap](/user-guide/roadmap) 문서를 확인해보세요.
