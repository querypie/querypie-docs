# Kubernetes Logs

### Overview

QueryPieでのすべての活動は監査ログに記録されます。ユーザーと管理者のログイン/ログアウト、権限付与と回収、Kubernetesに対するAPI実行履歴、リソースまたはセキュリティ設定変更など、QueryPieとアクセス制御対象リソースで発生するすべての活動履歴がログの対象です。リアルタイムで記録されるログについて、様々な条件で検索およびフィルタリングが可能です。

### 関連監査ログ種類

クエリパイ共通で提供される監査ログに加えて、クエリパイKubernetesアクセス制御（QueryPie KAC）ライセンス保有時、以下の監査ログの種類を提供します。

**1. General**

* User Access History : ユーザーログイン、ログアウト履歴
* Activity Logs : 管理者が実行するリソース登録および設定変更履歴
* Admin Role History : 管理者権限付与、変更、回収履歴

**2. Kubernetes**

* [Request Audit](kubernetes-logs/request-audit) : Kubernetes APIサーバー呼び出し内訳
* [Pod Session Recordings](kubernetes-logs/pod-session-recordings) : Pod Execなどの接続セッション内行為録画内訳
* [Kubernetes Role History](kubernetes-logs/kubernetes-role-history) : Kubernetesアクセス権限付与および回収履歴
