## Overview

QueryPieでのすべてのウェブアプリを監査ログに記録します。ユーザーと管理者のログイン/ログアウト、権限付与と回収、ウェブアプリケーションに対する実行履歴、リソースまたはセキュリティ設定変更など、QueryPieとアクセス制御対象リソースで発生するすべての活動履歴がログの対象です。リアルタイムで記録されるログについて、様々な条件で検索およびフィルタリングが可能です。

## 1. General

* Access Control Logs : アクセス制御関連ログ、ログインと履歴
* Policy Audit Logs : 管理者が実行するリソース登録および設定変更履歴
* Policy Exception Logs : 管理者権限付与、変更、回収履歴

## 2. Web Apps

* Web Access History : ウェブアプリケーションアクセス履歴内訳
* Web Event Audit : ウェブアプリケーションイベント監査内訳
* User Activity Recordings : ユーザー活動記録内訳
* Web App Role History : ウェブアプリケーションアクセス権限付与および回収履歴
* JIT Access Control Logs : JITアクセス制御ログ内訳
