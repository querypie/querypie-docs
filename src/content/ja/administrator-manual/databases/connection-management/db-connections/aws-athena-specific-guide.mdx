# AWS Athena専用ガイド

### コネクション手動作成

管理者ページ &gt; Databases &gt; Connection Management &gt; DB Connections &gt; Create Connections &gt; Athenaを選択します。その後必須の値を入力します。手動作成時、AWS Access KeyとAWS Secret Keyを入力する必要があります。この情報はIAMに登録された情報を使用する必要があります。AWS IAM &gt; Usersページで確認できます。

### Cloud Provider

管理者ページ &gt; Databases &gt; Connection Management &gt; Cloud Providerを通じて同期Athenaコネクションでは別途のAWS Access KeyとAWS Secret Key入力は必要ありません。
