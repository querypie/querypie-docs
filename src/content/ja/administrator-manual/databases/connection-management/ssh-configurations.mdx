# SSH Configurations

### Overview

SSH ConfigurationsはDBコネクション接続時に使用するSSHトンネリング情報を管理するページです。

<figure data-layout="center" data-align="center">
![Administrator &gt; Databases &gt; Connection Management &gt; SSH Configuration &gt; Create SSH Configuration](/administrator-manual/databases/connection-management/ssh-configurations/screenshot-20240730-230232.png)
<figcaption>
Administrator &gt; Databases &gt; Connection Management &gt; SSH Configuration &gt; Create SSH Configuration
</figcaption>
</figure>


1. Databases設定メニューでConnection Management &gt; SSH Configurationメニューに移動します。
2. 右上の`Create SSH`ボタンをクリックします。
3. SSHトンネリング登録のための情報を入力します。
   1. Name: SSHトンネリング構成名
   2. SSH Auth Type: 認証方式（PasswordまたはKey pair方式）
      * 選択肢に従ってモーダル下部認証情報入力内容が異なります
   3. SSH Host: SSH接続のためのHost情報（ドメイン、IP形態すべて入力可能）
   4. SSH Port: SSH接続のためのポート情報
   5. Username / Password : 接続時に使用するIDおよびパスワード
      * Password方式選択時表示
   6. SSH Private Key / SSH Key Passphrase : 接続時に使用する秘密鍵およびpassphrase情報
      * Key pair方式選択時表示
4. `Save`ボタンを押すと入力内容が保存されます。
