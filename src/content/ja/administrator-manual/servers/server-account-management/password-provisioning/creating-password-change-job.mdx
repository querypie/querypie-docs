# パスワード変更Job作成

### Overview

QueryPieに登録されたサーバーの接続アカウントパスワードを周期的に変更Jobを登録します。サーバーグループ単位でJobを作成し、パスワード変更はサーバーグループに属する個別サーバーを対象にアカウント別プロビジョニングを実行します。

<Callout type="info">
QueryPieを通じたサーバーアカウントのパスワード変更のためには、各サーバーにパスワード変更権限があるアカウントを登録しなければなりません。また、一つのJobを通じて複数のサーバーのアカウントパスワードを変更するためには、各サーバーに同じアカウント名とパスワードでJobで使用するサーバーアカウントを作成しなければなりません。
</Callout>


### パスワード変更Job作成

<figure data-layout="center" data-align="center">
![Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning &gt; Create Job](/administrator-manual/servers/server-account-management/password-provisioning/creating-password-change-job/image-20240831-162006.png)
<figcaption>
Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning &gt; Create Job
</figcaption>
</figure>

1. Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioningメニューに移動します
2. 右上の`+ Create Job`ボタンをクリックします。
3. Job作成のための以下の情報を入力します。
    1.  **Name** : パスワード変更JobのNameを入力します。
    2.  **Provisioning Information** : 実際のサーバーに接続してパスワード変更を実行するクレデンシャル情報を入力します。
        1.  **Account** : 実際のサーバーに接続してパスワード変更を実行するaccountを入力します。
        2.  **Password** : 実際のサーバーに接続してパスワード変更を実行するアカウントのPasswordを入力します。
    3.  **Job Frequency** : パスワード変更Jobの反復周期を設定します。手動変更または周期的な変更を設定できます。
    4.  **Provisioning Target** : パスワードを変更するサーバーがグループ化されたServer Groupを選択します。Server Groupの各個別サーバーに **** 接続して、各アカウントのパスワードを変更します。サーバーアカウントのパスワードを周期的に変更するためには、Server GroupsのAccounts項目の「Provisioning」項目がOnになっていなければなりません。
4. `Save`ボタンを通じて保存します。

<Callout type="info">
Provisioning Informationアカウントのパスワードを周期的に変更するためには、Server Groupにアカウントが登録されていなければなりません。
</Callout>

1. Administrator &gt; Servers &gt; Server Account Management &gt; Password Provisioning &gt; Active Directoryタブに移動します
2. 右上の`+ Create Job`ボタンをクリックします。
3. Job作成のための以下の情報を入力します。
    1.  **Name** : パスワード変更JobのNameを入力します。
    2.  **Target Directory** : パスワード変更を実行しようとするADを選択します。  **** 
    3.  **Job Frequency** : パスワード変更Jobの反復周期を設定します。手動変更または周期的な変更を設定できます。
    4.  **Provisioning Target** : パスワードを変更するUPNを選択します。
4. `Save`ボタンを通じて保存します。
