# Server Account Templates

### Overview

コネクションに接続できるアカウントを事前に登録してテンプレート形態で保存しておけます。このように事前に作成したテンプレートはサーバーまたはサーバーグループで簡単で便利に呼び出せます。


### サーバーアカウントテンプレート作成

<figure data-layout="center" data-align="center">
![Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templates &gt; Create Template](/administrator-manual/servers/server-account-management/server-account-templates/image-20240828-011910.png)
<figcaption>
Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templates &gt; Create Template
</figcaption>
</figure>

1. Administrator &gt; Servers &gt; Server Account Management &gt; Server Account Templatesメニューに移動します。
2. 右上の`Create Template`ボタンをクリックします。
3. テンプレート作成のための以下の情報を入力します。
    1.  **Name** : テンプレートを区別するための名前を入力します。
    2.  **Description** : テンプレートを説明できる内容を入力します。
4. `Save`ボタンを通じて保存します。
5. リストで新しく作成されたテンプレートをクリックします。
6. Account項目で`Add Account`をクリックしてテンプレートに登録するアカウントを追加します。
    * 個別Account設定方法は[サーバーをグループで管理](../connection-management/server-groups/managing-servers-as-groups)を参考にします。
7. 右上の`Save Changes`をクリックして保存します。
