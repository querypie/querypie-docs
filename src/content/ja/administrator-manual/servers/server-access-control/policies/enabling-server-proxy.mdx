# Server Proxy使用有効化

### Overview

サーバー接続時Proxy接続をサポートしています。基本的にQueryPieで提供するウェブTerminalを通じてサーバーに接続でき、QueryPieでAgentを通じて既存のOS Account/passwordで使用していたツールからサーバーに接続できます。ウェブTerminalと一緒にProxy接続をサポートして多様なユーザー環境でも問題なくサーバーアクセスを制御し、ポリシーを適用でき、ログを残せます。現在Proxy接続は（Mac）Iterm2、Terminal、（Windows）Windows Console Host、Windows Terminal、PuTTYをサポートしています。


### Proxy使用オプション有効化

Proxyを通じたサーバー接続を許可するためには、まずProxy Usageが有効化されたPolicyを作成しなければなりません。Policyを作成する方法およびProxy Usageを有効化する方法は以下のドキュメントを参考にしてください。

* [サーバーアクセスポリシー設定](setting-server-access-policy)
