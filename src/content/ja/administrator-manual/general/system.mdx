## Overview

QueryPieのシステム設定を管理するメニューです。各サブメニューで外部セキュリティサービスとの連携、External API Token、QueryPie内部で定期的に実行される作業を管理できます。

## サブメニュー

それぞれのメニューは以下の機能を提供します。

* [Integrations](system/integrations) : SIEMサービスおよびSecret Store連携管理
* [API Token](system/api-token) : External API Token管理
* [Jobs](system/jobs) : QueryPie内部の定期的/手動作業一覧

詳細については、各サブメニューの説明をご参照ください。
