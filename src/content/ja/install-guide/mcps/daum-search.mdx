# Daum Search

Daum検索APIを通じて、ウェブ文書、画像、ブログ、カフェなどの検索機能を使用できます。

## 主な機能
- ウェブ文書検索
- 画像検索
- ブログ投稿検索
- カフェ投稿検索

## 基本情報

### 認証方式
API Keyを使用して認証します。

### 入力パラメータ
- **API Key**（必須）：Kakao開発者センターで発行されたREST API キー

## サポートするMCPツール

### 検索ツール
#### search_web
Daumでウェブ文書を検索します。

**入力パラメータ：**
- `query`（必須）：検索クエリ
- `sort`（オプション）：結果文書のソート方式（accuracy、recency）
- `page`（オプション）：結果ページ番号（1~50）
- `size`（オプション）：1ページに表示する文書数（1~50）

#### search_image
Daumで画像を検索します。

**入力パラメータ：**
- `query`（必須）：検索クエリ
- `sort`（オプション）：結果文書のソート方式（accuracy、recency）
- `page`（オプション）：結果ページ番号（1~50）
- `size`（オプション）：1ページに表示する文書数（1~80）

#### search_blog
Daumでブログ投稿を検索します。

**入力パラメータ：**
- `query`（必須）：検索クエリ
- `sort`（オプション）：結果文書のソート方式（accuracy、recency）
- `page`（オプション）：結果ページ番号（1~50）
- `size`（オプション）：1ページに表示する文書数（1~50）

#### search_cafe
Daumでカフェ投稿を検索します。

**入力パラメータ：**
- `query`（必須）：検索クエリ
- `sort`（オプション）：結果文書のソート方式（accuracy、recency）
- `page`（オプション）：結果ページ番号（1~50）
- `size`（オプション）：1ページに表示する文書数（1~50）

## 使用例

例えば、
- プロンプトで「最新のAI技術に関するブログ記事を探して」と入力すると、
- LLMはMCPを通じて`search_blog`ツールを呼び出し、AI技術関連のブログ記事を検索します。
- そして、ツール呼び出し結果に基づいてチャットで応答します。

## API Key設定方法
1. Kakao開発者センターでアプリケーションを登録します。
2. REST API キーを発行します。
3. QueryPie AIでKakao Search統合を選択します。
4. 発行されたAPI キーを入力して接続を完了します。

## 注意事項
- 生成したAPI Keyは安全に保管してください。
- 1日の呼び出し制限を確認し、適切に使用してください。
- 検索結果はDaum検索エンジンの結果に基づいています。
- 著作権のあるコンテンツ使用時は注意してください。
