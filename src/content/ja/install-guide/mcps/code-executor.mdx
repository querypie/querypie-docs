# Code Executor

サンドボックス環境でLLMが生成したコードを安全に実行することができます。

## 主要機能
- サンドボックス環境での安全なコード実行
- 実行結果・エラーメッセージ提供
- リアルタイムコードテスト

### 使用可能な機能
- 基本JavaScript文法と内蔵オブジェクト（Array、Object、Math、Dateなど）
- 関数定義・呼び出し
- 変数宣言・操作
- 条件文、繰り返し文などの制御構造
- 基本的な計算・データ処理

### 活用例
- 数学計算やアルゴリズムテスト
- データ変換・処理ロジック検証
- 簡単な関数やクラス実装テスト

## 基本情報

### 認証方式
別途認証は必要ありません。

### 入力パラメータ
追加で入力するパラメータはありません。

## サポートするMCPツール

### run_code
サンドボックス環境でJavaScript（ECMAScript）コードを実行し結果を返します。

## 主要制約事項
- 外部ライブラリimport不可 - npmパッケージや外部モジュールを取得できません。
- ファイルシステムアクセス不可 - ファイル読み取り/書き込み作業は不可能です。
- ネットワークリクエスト制限 - fetch、XMLHttpRequestなどを通じた外部API呼び出しは不可能です。
- DOM操作不可 - document、windowオブジェクトなどのブラウザAPI使用は不可能です。
- 非同期作業制限 - setTimeout、setIntervalなどのタイマー関数使用は不可能です。
