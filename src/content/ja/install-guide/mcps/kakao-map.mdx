# Kakao Map

Kakao地図 APIを通じて、場所検索、住所変換、座標変換などの地図関連機能を使用できます。

## 主な機能
- キーワードベースの場所検索
- カテゴリ別場所検索
- 住所を座標に変換
- 座標を住所に変換
- 座標を行政区域情報に変換
- 座標系間変換

## 基本情報

### 認証方式
API Keyを使用して認証します。

### 入力パラメータ
- **API Key**（必須）：Kakao開発者センターで発行されたREST API キー

## サポートするMCPツール

### 場所検索ツール
#### search_places
キーワードを基にKakao地図で場所を検索します。

**入力パラメータ：**
- `query`（必須）：検索キーワード
- `x`（オプション）：中心座標のX値（経度）
- `y`（オプション）：中心座標のY値（緯度）
- `radius`（オプション）：中心座標からの半径距離（メートル）
- `page`（オプション）：結果ページ番号（1~45）
- `size`（オプション）：1ページに表示する文書数（1~15）

#### map_category_search
カテゴリ別にKakao地図で場所を検索します。

**入力パラメータ：**
- `category_group_code`（必須）：カテゴリグループコード
- `x`（オプション）：中心座標のX値（経度）
- `y`（オプション）：中心座標のY値（緯度）
- `radius`（オプション）：中心座標からの半径距離（メートル）
- `page`（オプション）：結果ページ番号
- `size`（オプション）：1ページに表示する文書数

### 座標変換ツール
#### map_address_to_coord
住所を座標に変換します。

**入力パラメータ：**
- `query`（必須）：住所検索語

#### map_coord_to_address
座標を住所に変換します。

**入力パラメータ：**
- `x`（必須）：X座標（経度）
- `y`（必須）：Y座標（緯度）

#### map_coord_to_region_code
座標を韓国行政区域情報に変換します。

**入力パラメータ：**
- `x`（必須）：X座標（経度）
- `y`（必須）：Y座標（緯度）

#### map_trans_coord
座標系間で座標を変換します。

**入力パラメータ：**
- `x`（必須）：X座標
- `y`（必須）：Y座標
- `input_coord`（必須）：入力座標系
- `output_coord`（必須）：出力座標系

## 使用例

例えば、
- プロンプトで「江南駅近くのカフェを探して」と入力すると、
- LLMはMCPを通じて`search_places`ツールを呼び出し、江南駅近くのカフェを検索します。
- そして、ツール呼び出し結果に基づいてチャットで応答します。

## API Key設定方法
1. Kakao開発者センターでアプリケーションを登録します。
2. REST API キーを発行します。
3. QueryPie AIでKakao Map統合を選択します。
4. 発行されたAPI キーを入力して接続を完了します。

## 注意事項
- 生成したAPI Keyは安全に保管してください。
- 1日の呼び出し制限を確認し、適切に使用してください。
- 座標はWGS84座標系を基本として使用します。
