# 🧩 MCPサーバー連携

MCP（Model Context Protocol）は、AIモデルが外部ツールに接続する方式を標準化するプロトコルです。

QueryPie AI Hubでは独自にMCPサーバーを提供しており、これを通じて様々なツールやデータソースに安全に接続することができます。

## ワークフロー自動化
ワークフロー自動化ツールを連携し、順次思考ツールを活用して複雑な問題を解決することができます。

- [Sequential Thinking](./mcps/sequentialthinking) - 順次思考ツール
- [n8n Chat Trigger Node](./mcps/n8n-chat-trigger-node) - n8nワークフロートリガー
- [Dify API Access](./mcps/dify-api-access) - Dify AIプラットフォーム連携
