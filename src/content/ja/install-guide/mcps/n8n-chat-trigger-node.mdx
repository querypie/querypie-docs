# N8n Chat Trigger Node

N8nのChat Triggerノードを通じてワークフローをトリガーし、チャットボット及びその他のチャットインターフェースを呼び出すことができます。

## 主要機能
- N8nワークフロートリガー
- チャットベースワークフロー実行
- セッションベース対話管理
- 外部チャットボット連携

## 基本情報

### 認証方式
Basic Authenticationを使用して認証します。

### 入力パラメータ
- **Chat URL** (必須): N8n Chat TriggerノードのWebhook URL
- **Use Auth** (選択): 認証使用可否
- **Basic Auth Username** (選択): Basic認証ユーザー名
- **Basic Auth Password** (選択): Basic認証パスワード

## サポートするMCPツール

### ワークフロートリガーツール
#### trigger_chat_trigger_node
Chat Triggerノードをトリガーしてチャットボット及びその他のチャットインターフェースを呼び出します。

**入力パラメータ:**
- `chatInput` (必須): チャット入力メッセージ

## 使用例

例えば、
- Promptで「n8nワークフローに『こんにちは』というメッセージを送ってください」と入力すると、
- LLMはMCPを通じて`trigger_chat_trigger_node`ツールを呼び出してN8nワークフローを実行します。
- そして、Tool呼び出し結果を基にチャットで応答します。

## 設定方法
1. N8nでChat Triggerノードが含まれたワークフローを作成します。
2. Chat TriggerノードのWebhook URLをコピーします。
3. QueryPie AIでN8n Chat Trigger Node統合を選択します。
4. Webhook URLと必要な認証情報を入力して接続を完了します。

## 注意事項
- Webhook URLは安全に保管してください。
- Basic認証を使用する場合、ユーザー名とパスワードを安全に管理してください。
- n8nワークフローが有効化されているか確認してください。
- セッションIDは自動的に生成され、対話コンテキストを維持します。
