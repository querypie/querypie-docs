# 🛠️ LLMアプリケーション

QueryPie AI Hubは様々なLLMアプリケーションをサポートしています。<br/>
MCPプリセット別に提供されるローカルプロキシインストールコマンドを実行すると、希望するLLMアプリケーションにインストールできます。

## サポートされているLLMアプリケーション

- Cursor
- Claude Desktop
- Windsurf

## 使用方法

![3rd Party Installation](@/app/img/3rd-party.png)

1. MCPプリセットを選択します。
2. InstallationタブでGenerate Commandボタンをクリックします。
    - 生成されたローカルプロキシインストールコマンドは10分間有効で、1回のみ使用できます。
3. ターミナルを開き、ローカルプロキシインストールコマンドを実行します。
4. ローカルプロキシインストールが完了すると、LLMアプリケーションでQueryPie AI HubのMCPプリセットを使用できます。

## 注意事項

- インストールコマンドは10分間有効で、1回のみ使用できます。
- 実際の認証に使用されるCLI TokenはMCP Configurationに保存されます。CLI Tokenが流出しないよう注意してください。
  - 発行されたCLI Tokenは、プロフィール > 設定 > CLI Tokenメニューで削除できます。
- MCPプリセットのInstructionは3rd Party LLMアプリケーションでは動作しません。
- 最大連携可能なツールの数は、LLMアプリケーションの制約事項によります。
    - Cursor: 40個
    - Claude Desktop: 10個
    - Windsurf: 100個
