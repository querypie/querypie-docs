### QueryPie 10.0.0 Release

2024年07月22日


**New Features**

* [General] ユーザー/管理者ページ分離提供 
* [General] ユーザーダッシュボード(My Dashboard)メニュー新設 
* [Security] パスワード再使用禁止回数設定機能提供
* [AIDD] AI Data Discovery機能新規リリース 
* [Workflow] Access Role Request新規提供 
* [DAC] Connectionページ内Tag機能提供
* [DAC] GCP同期再サポート 
* [DAC] Hive Kerberos認証サポート 
* [DAC] SQLServer SSMS Proxy接続サポート
* [SAC] Windowsサーバーアクセス制御(RDP)サポート


**Enhancement**

* [SCIM] ユーザーおよびグループImport機能追加およびGroup Push機能改善 
* [Workflow] SQL Requestでクエリ実行後停止(Stop)できる機能提供
* [Audit] Audit Log Export目録でTask別状態値確認可能に改善 
* [Audit] Audit Log ExportでWorkflow SQL Requestタイプ抽出時ID値追加提供
* [DAC] SQL Editorで複数コネクション接続時セッションを一括終了できる機能提供


**Bug Fix**

* [SCIM] Profile EditorによるAttribute更新異常動作イシュー解決
* [External API] 9.19.0アップグレード後API Token権限Scopes値がSelect Roleで付与される問題解決
* [DAC] SQL Server特定クエリ(IF/EXEC)が実行されない問題修正
* [DAC] ブラウザ新規タブでConnection接続時間欠的エラー発生する問題修正
* [Activity Log] null値が含まれた過去データ照会時エラー発生する問題解決 
* [Security] QueryPie Web IP Access Control内特定値設定時エラー発生する問題解決
* [Agent] MariaDBコネクションをOpen Connection With機能で接続時エラー発生する問題解決 
* [DAC] MariaDB Temporary TableのDMLSnapshot値がOnの場合正常動作しないバグ修正
* [Agent] UbuntuでFirefoxにAgent接続不可な問題解決


**External API**

* Workflow - Access Role Request Details照会API追加
